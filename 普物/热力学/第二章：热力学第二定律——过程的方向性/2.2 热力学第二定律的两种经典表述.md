## 2.2 热力学第二定律的两种经典表述

在 2.1 节中，我们建立了第二定律的物理直觉——自然界中的过程是有方向的，不是所有满足能量守恒的过程都能发生。现在我们要把这种直觉转化为精确的表述。

历史上，热力学第二定律有两种最著名的表述：开尔文-普朗克表述和克劳修斯表述。它们表面上看起来完全不同——一个关于热机，一个关于热传导——但它们是等价的。违反其中一种表述，必然导致违反另一种。这种等价性不是偶然的，它揭示了第二定律的深层统一性：一切自然过程的方向性，都可以归结为同一原理。

这一节我们从两种表述的物理含义出发，然后用反证法严格证明它们的等价性。

---

### 一、开尔文-普朗克表述——第二类永动机不可能

**开尔文-普朗克表述**（Kelvin-Planck statement）：

> 不可能从单一热源吸热，使之完全转化为功，而不产生其他影响。

这个表述的历史背景是“第二类永动机”（perpetual motion machine of the second kind）——一种假想的机器，它从海洋、大气或地球内部等单一热源吸收热量，并全部转化为功，从而提供无穷无尽的能量。

**开尔文表述的精确含义**：

“单一热源”指所有参与过程的热源具有相同的温度。如果存在两个不同温度的热源，那么热量从高温热源流向低温热源的过程中可以部分转化为功——这是热机的工作原理。开尔文表述说的是：只有一个热源时，不可能把热量完全转化为功。

“不产生其他影响”是关键词。如果在转化过程中同时产生了其他变化（如气体膨胀、化学反应、相变等），那么热量可以完全转化为功。例如，理想气体等温膨胀：气体从单一热源吸收热量，全部转化为对外做的功。但在这个过程中，气体的体积发生了变化——这就是“其他影响”。如果要求气体回到初始状态（循环过程），就必须压缩气体，这需要外界做功。因此，在循环过程中，不可能从单一热源吸热完全转化为功。

开尔文表述的另一种说法是：**任何循环工作的热机，都必须至少向一个低温热源放热。**

**第二类永动机与第一类永动机的区别**：

| | 第一类永动机 | 第二类永动机 |
|--|-------------|-------------|
| 违反的定律 | 热力学第一定律（能量守恒） | 热力学第二定律（方向性） |
| 特征 | 不消耗能量而做功 | 从单一热源吸热完全转化为功 |
| 否定方式 | 能量守恒 | 方向性限制 |

第一类永动机被第一定律否定，第二类永动机被第二定律否定。第二类永动机并不违反能量守恒——如果它存在，它从海洋中提取热量，海洋温度略微下降，能量守恒是满足的。但它违反了方向性：热量不能完全转化为功而不产生其他影响。


### 二、克劳修斯表述——热传导的方向性

**克劳修斯表述**（Clausius statement）：

> 热量不可能自发地从低温物体传向高温物体，而不产生其他影响。

这个表述直接对应我们在 2.1 节中讨论的热传导方向性。热量从高温传到低温是自发的、自然的；反过来不行。

“自发”的含义：不需要外界对系统做功，不消耗任何外部能量。如果借助制冷机，热量可以从低温传到高温——但这是以消耗功为代价的，不是“自发”的，而且产生了“其他影响”（制冷机的工作过程）。

“不产生其他影响”同样是关键词。如果同时产生其他变化，热量可以从低温传到高温。例如，热泵利用外界做功把热量从低温环境“泵”到高温环境，这个过程产生了“其他影响”——消耗了功。克劳修斯表述不禁止这种非自发过程，它只禁止没有其他影响的自发过程。

克劳修斯表述的另一种说法是：**在没有外界干预的情况下，热量总是从高温物体流向低温物体，直到两者温度相等。**


### 三、两种表述等价性的严格证明

现在我们用反证法证明：开尔文-普朗克表述与克劳修斯表述是等价的。

#### 方向一：克劳修斯表述不成立 ⇒ 开尔文表述不成立

**【假设】** 克劳修斯表述不成立。即存在一个自发过程，热量 $Q_c$ 可以自发地从低温热源 $T_c$ 传递到高温热源 $T_h$，而不产生任何其他影响。

**【构造】** 安排一个热机，工作于高温热源 $T_h$ 和低温热源 $T_c$ 之间：

- 热机从高温热源吸收热量 $Q_h$
- 对外输出功 $W$
- 向低温热源排放热量 $Q_c$

其中热机效率 $\eta = W/Q_h$，且由能量守恒 $Q_h = Q_c + W$，所以 $W = Q_h - Q_c$。

**【组合】** 将上述两个装置组合：

| 装置 | 操作 | 热量交换 |
|------|------|----------|
| **热机** | 从 $T_h$ 吸热 $Q_h$，向 $T_c$ 放热 $Q_c$，输出功 $W$ | $T_h$ 失去 $Q_h$，$T_c$ 得到 $Q_c$ |
| **假设装置** | 将热量 $Q_c$ 从 $T_c$ 自发泵回 $T_h$ | $T_c$ 失去 $Q_c$，$T_h$ 得到 $Q_c$ |
| **净效果** | | $T_h$ 净失去 $Q_h - Q_c$，$T_c$ 无净变化，输出功 $W$ |

**【结论】** 组合装置的净效果是：从单一热源 $T_h$ 吸收热量 $Q_h - Q_c$，并全部转化为功 $W = Q_h - Q_c$，没有产生任何其他影响。这就违反了开尔文-普朗克表述。

```
┌──────────────────────────────────────────────────────────────┐
│                        能量流向示意图                          │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│   高温热源 T_h                                                │
│        │                                                     │
│        │ Q_h（吸热）                                          │
│        ▼                                                     │
│    ┌─────────┐                                               │
│    │  热 机   │───► 输出功 W = Q_h - Q_c                      │
│    └─────────┘                                               │
│        │                                                     │
│        │ Q_c（放热）                                          │
│        ▼                                                     │
│   低温热源 T_c                                                │
│        │                                                     │
│        │ Q_c（自发逆传！——来自克劳修斯假设）                     │
│        └──────────────────────┐                              │
│                               ▼                              │
│   回到高温热源 T_h（形成一个循环）                               │
│                                                               │
│   ──────────────────────────────────────────────────          │
│   净效果：T_h 失去 Q_h - Q_c，全部转化为功 W                     │
│   → 违反开尔文表述！                                            │
└───────────────────────────────────────────────────────────────┘
```

因此：**克劳修斯表述的违反 ⇒ 开尔文表述的违反**。


#### 方向二：开尔文表述不成立 ⇒ 克劳修斯表述不成立

**【假设】** 开尔文表述不成立。即存在一个热机，可以从单一热源（温度为 $T$）吸收热量 $Q$，并完全转化为功 $W = Q$，不产生任何其他影响。

**【构造】** 用这个热机输出的功 $W$ 来驱动一个制冷机。制冷机工作于低温热源 $T_c$ 和高温热源 $T_h$ 之间：

- 从低温热源吸收热量 $Q_c$
- 向高温热源释放热量 $Q_h$
- 消耗功 $W$

由制冷机的能量守恒：$Q_h = Q_c + W$。

**【组合】** 将上述两个装置组合：

| 装置 | 操作 | 热量交换 |
|------|------|----------|
| **假设热机** | 从单一热源（设为 $T_h$）吸热 $Q$，完全转化为功 $W=Q$ | $T_h$ 失去 $Q$ |
| **制冷机** | 输入功 $W$，从 $T_c$ 吸热 $Q_c$，向 $T_h$ 放热 $Q_h = Q_c + W$ | $T_c$ 失去 $Q_c$，$T_h$ 得到 $Q_c + W$ |
| **净效果** | | $T_h$ 净得到 $Q_c$（因为得到 $Q_c+W$，失去 $Q$，且 $W=Q$），$T_c$ 失去 $Q_c$ |

**【结论】** 组合装置的净效果是：热量 $Q_c$ 从低温热源 $T_c$ 自发地传递到了高温热源 $T_h$，没有产生其他影响。这就违反了克劳修斯表述。

```
┌──────────────────────────────────────────────────────────────┐
│                        能量流向示意图                          │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│   单一热源 T_h                                                │
│        │                                                     │
│        │ Q（吸热）                                            │
│        ▼                                                     │
│    ┌─────────┐                                               │
│    │假设热机  │───► 输出功 W = Q（完全转化！）                  │
│    └─────────┘                                               │
│        │                                                     │
│        │ 功 W 驱动                                            │
│        ▼                                                     │
│    ┌─────────┐                                               │
│    │ 制冷机  │◄─── 从 T_c 吸热 Q_c                            │
│    └─────────┘                                               │
│        │                                                     │
│        │ Q_h = Q_c + W = Q_c + Q（向 T_h 放热）               │
│        ▼                                                     │
│   回到高温热源 T_h                                            │
│                                                              │
│   ──────────────────────────────────────────────────         │
│   净效果：T_c 失去 Q_c，T_h 得到 Q_c                           │
│   → 违反克劳修斯表述！                                         │
└──────────────────────────────────────────────────────────────┘
```

因此：**开尔文表述的违反 ⇒ 克劳修斯表述的违反**。

**等价性结论**：

方向一和方向二共同证明：开尔文-普朗克表述和克劳修斯表述在逻辑上是等价的——如果其中一个不成立，另一个也不成立。既然它们在实验上（在自然界中）都成立，它们描述的实际上是同一条物理定律的两个侧面。


### 四、第二定律的统一本质

两种表述的等价性揭示了一个深刻的事实：**热力学第二定律是关于自然过程方向性的一条统一原理**。

开尔文表述说的是：功可以完全转化为热（摩擦、焦耳实验），但热不能完全转化为功（热机有极限效率）。

克劳修斯表述说的是：热量可以自发地从高温传到低温，但不能自发地从低温传到高温。

这两者听起来不同，但它们是同一原理的两种表现。这个统一原理就是：**孤立系统的熵永不减少**（将在 2.4 节中严格定义）。热量从高温传到低温是熵增过程；热完全转化为功不是熵增过程（实际上是熵减过程，除非伴随其他变化）。

**等价性证明的逻辑结构**：

```
┌─────────────────────────────────────────────────────────────┐
│                      等价性证明的逻辑结构                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│       开尔文-普朗克表述                                       │
│       （热不能完全转化为功）                                   │
│              │                                               │
│              │  反证法方向二                                  │
│              │  若开尔文不成立 → 克劳修斯不成立                 │
│              ▼                                               │
│       克劳修斯表述                                            │
│       （热量不能自发从低温到高温）                              │
│              │                                               │
│              │  反证法方向一                                  │
│              │  若克劳修斯不成立 → 开尔文不成立                 │
│              ▼                                               │
│       两者等价！同一物理定律的两种表述                           │
│                                                              │
│       本质：自然过程的方向性 → 孤立系统熵永不减少                 │
└──────────────────────────────────────────────────────────────┘
```

**第二定律与第一定律的关系**：

- 第一定律说：能量守恒，功和热量等价（可以相互转化，总量不变）。
- 第二定律说：能量转化有方向，热量转化为功有上限。

打个比方：
- 第一定律：你账户里的钱不能凭空增加或减少（总量守恒）。
- 第二定律：你不能把所有的现金都变成盈利——总有一部分必须作为成本“浪费”掉。或者说，你可以把功全部转化为热（就像把钱全部花掉），但不能把热全部转化为功（就像不能把花掉的钱全部赚回来）。


### 五、第二类永动机——从海洋中提取能量

开尔文表述最著名的推论是：第二类永动机不可能。

为什么我们无法从海洋中提取能量？海洋含有巨大的热量（温度约 $300\,\text{K}$），如果能把海洋中的热量提取出来并转化为功，人类将拥有无穷无尽的能源。但第二定律告诉我们：不可能从单一热源吸热完全转化为功。我们必须同时有一个低温热源（如极地海水）来接收排放的热量。由于海洋和极地之间的温差很小，卡诺效率 $\eta = 1 - T_c/T_h$ 非常低（约 $5\%$），而且需要巨大的热交换器——这使得海洋热能转换在经济上不可行。

第二类永动机不违反能量守恒，但它违反方向性。这正是它和第一类永动机的根本区别。


### 六、等价性证明的深层含义

两种表述的等价性证明依赖于一个关键事实：它们都涉及“不产生其他影响”这个条件。

- 如果热量能从低温传到高温而不产生其他影响，那么我们就可以用它来“修补”热机向低温的放热，从而把热机的效率提高到 $100\%$ ——从单一热源吸热完全转化为功。
- 如果功可以完全从单一热源产生，那么我们就可以用它来驱动制冷机，把热量从低温泵到高温——从而实现了自发逆传热。

这个循环论证说明了第二定律的**统一性**：方向性的各种表现形式——热传导的方向、功热转化的方向、扩散的方向——都是同一原理的不同侧面。违反一种方向性，必然导致所有方向性都被违反。


### 七、总结

把 2.2 节的全部内容压缩成几条线：

1. **开尔文-普朗克表述**：不可能从单一热源吸热完全转化为功而不产生其他影响。第二类永动机不可能。

2. **克劳修斯表述**：热量不可能自发地从低温物体传向高温物体而不产生其他影响。

3. **等价性证明**（反证法，两种方向）：
   - **方向一**（克劳修斯⇒开尔文）：若热量可从低温自发传到高温，用它“修补”热机向低温的放热 → 热机从单一热源吸热完全转化为功 → 违反开尔文表述。
   - **方向二**（开尔文⇒克劳修斯）：若存在热机从单一热源吸热完全转化为功，用此功驱动制冷机 → 热量从低温传到高温 → 违反克劳修斯表述。

4. **第二定律的统一本质**：两种表述是同一原理（自然过程的方向性）的两个侧面。统一表述：孤立系统的熵永不减少。

5. **第二类永动机**：不违反能量守恒，但违反方向性。从海洋提取能量的尝试被第二定律否定。

6. **第一定律与第二定律的关系**：第一定律说能量守恒，第二定律说能量转换有方向。两者共同构成热力学的完整基础。

两种表述的等价性证明不仅是逻辑练习，它揭示了第二定律的普适性：无论你从哪个角度观察自然过程的方向性——热传导、功热转化、扩散——你都会得到同一个结论。这个结论将在 2.4 节中被精炼为一个数学量——熵。有了熵，第二定律将从一个定性的方向性描述，变为一个精确的、可计算的不等式：$\Delta S \ge 0$。
