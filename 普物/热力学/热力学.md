# 热力学

> **课程定位**：大学物理五门基础课（力学、热学、光学、电磁学、原子物理）中的第二门。从宏观经验定律出发建立严密的逻辑体系，在适当位置引入分子动理论作为微观解释。

## 第零章：数学预备知识与基本概念（M.1 - M.5）

### M.1 偏导数与全微分 [M.1 偏导数与全微分 - 知乎](https://zhuanlan.zhihu.com/p/2062559913633698674)
- **核心任务**：掌握偏导数的定义、全微分形式、混合偏导数的交换性——热力学中大量使用这些工具。
- 偏导数的定义与计算：$\left(\frac{\partial f}{\partial x}\right)_y$ 的下标含义——明确哪些变量在偏导过程中保持不变
- 全微分：$df = \left(\frac{\partial f}{\partial x}\right)_y dx + \left(\frac{\partial f}{\partial y}\right)_x dy$
- 恰当微分与非恰当微分的区别——状态函数 vs 过程量
- 混合偏导数可交换的条件：$\frac{\partial^2 f}{\partial x\partial y} = \frac{\partial^2 f}{\partial y\partial x}$（恰当微分的判据）
- 链式法则的推广——热力学中变量之间的隐式关系
- 循环关系：$\left(\frac{\partial x}{\partial y}\right)_z \left(\frac{\partial y}{\partial z}\right)_x \left(\frac{\partial z}{\partial x}\right)_y = -1$ ——推导与证明
- 雅可比行列式在热力学变量变换中的应用

**推导目标**：
- 从全微分条件推导恰当微分判据
- 证明循环关系

### M.2 理想气体状态方程 [M.2 理想气体状态方程 - 知乎](https://zhuanlan.zhihu.com/p/2062559974912528631)

- 查理定律（$V/T = \text{常数}$，$p$ 恒定）与盖-吕萨克定律（$p/T = \text{常数}$，$V$ 恒定）
- 理想气体状态方程：$pV = nRT$
- 摩尔、阿伏伽德罗常数 $N_A = 6.022\times 10^{23}\,\text{mol}^{-1}$、普适气体常数 $R = 8.314\,\text{J/mol·K}$ 的物理意义
- 分子数密度表述：$p = nkT$，$k = R/N_A = 1.38\times 10^{-23}\,\text{J/K}$（玻尔兹曼常数）
- 理想气体的微观模型——分子动理论的三条基本假设
- 理想气体的适用范围与局限性

**推导目标**：
- 从三条实验定律导出理想气体状态方程

## 第一章：热力学第一定律——能量守恒的宏观表述（1.1 - 1.8）

> **核心问题**：热量和功之间是什么关系？能量守恒在热现象中如何精确表达？

### 1.1 功的广义定义——热力学中的功 [1.1 功的广义定义——热力学中的功 - 知乎](https://zhuanlan.zhihu.com/p/2062560579093521079)
- **核心任务**：将“功”的概念从力学推广到热力学，理解体积功的核心地位。
- 体积功的严格定义：$dW = p_{\text{ext}} dV$ ——系统对外做功为正的约定
- 准静态过程中的体积功：$dW = p\,dV$（系统内部压强等于外部压强）
- 功的几何意义：$p$-$V$ 曲线下的面积即为功
- 不同路径下功的大小不同——功是过程量的直接体现
- 非体积功简介：电功（电池）、表面功（液膜）、磁功（磁化）

**推导目标**：
- 从力学功的定义 $dW = \mathbf{F}\cdot d\mathbf{r}$ 导出体积功 $dW = p dV$
- 推导不同过程中功的表达式

### 1.2 热量——热力学中的另一种能量传递方式 [1.2 热量——热力学中的另一种能量传递方式 - 知乎](https://zhuanlan.zhihu.com/p/2062560601847632260)
- **核心任务**：建立“热量”作为能量传递形式的严格概念。

- 焦耳热功当量实验的历史意义
- 热容量与比热容的定义：$C = \delta Q/dT$，$c = C/m$，$C_m = C/n$
- 定容热容 $C_V$ 与定压热容 $C_p$ 的物理区别——对应不同约束条件下的热响应
- 热量的传递方式：传导、对流、辐射（定性区分）

**推导目标**：
- 理解热功当量的物理意义
- 区分热容量、比热容、摩尔热容三个概念

### 1.3 热力学第一定律的建立 [1.3 热力学第一定律的建立 - 知乎](https://zhuanlan.zhihu.com/p/2062560635725026663)
- **核心任务**：建立能量守恒在热现象中的精确表述，理解其普适性。
- 第一定律的表述：系统吸收的热量等于内能增量加对外做功
- 数学形式：$Q = \Delta U + W$（系统吸热为正，对外做功为正）
- 微分形式：$dU = \delta Q - p\,dV$（准静态过程）
- 内能 $U$ 是状态函数——与路径无关，由系统的状态唯一决定
- 第一定律的循环过程表述：$\oint \delta Q = \oint \delta W$（循环过程中吸热等于做功）
- 第一类永动机的不可能性——不消耗能量而持续做功的机器违反第一定律
- 热力学第一定律作为能量守恒定律在热现象中的具体形式

**推导目标**：
- 从能量守恒导出第一定律的数学形式

### 1.4 内能——微观能量的宏观表现 [1.4 内能——微观能量的宏观表现 - 知乎](https://zhuanlan.zhihu.com/p/2062560655039903457)

- 内能的广延性——与物质的量成正比

### 1.5 等值过程——第一定律的系统应用 [1.5 等值过程——第一定律的系统应用 - 知乎](https://zhuanlan.zhihu.com/p/2062560673792537450)
- **核心任务**：计算四种基本热力学过程中的功、热量和内能变化，建立标准计算流程。
- **等容过程**（$V = \text{常数}$）：
  - 做功为零：$W = 0$
  - 热量等于内能变化：$Q_V = \Delta U = nC_V\Delta T$
- **等压过程**（$p = \text{常数}$）：
  - 做功：$W = p\Delta V$
  - 热量：$Q_p = \Delta U + p\Delta V = nC_p\Delta T$
  - 引入焓的方便性
- **等温过程**（$T = \text{常数}$，理想气体）：
  - 内能不变：$\Delta U = 0$
  - 做功与热量：$W = Q = nRT\ln(V_2/V_1)$
- **绝热过程**（$Q = 0$）：
  - 功等于内能减少：$W = -\Delta U$
  - 推导 $pV^\gamma = \text{常数}$，$TV^{\gamma-1} = \text{常数}$
  - 绝热指数 $\gamma = C_p/C_V$
  - 绝热过程与等温过程的对比—— $p$-$V$ 图上绝热线比等温线更陡

**推导目标**：
- 推导理想气体在四种等值过程中的功、热、内能变化
- 推导绝热过程方程 $pV^\gamma = \text{常数}$
- 证明 $\gamma > 1$，绝热线比等温线陡

### 1.7 焓——等压过程的自然变量 [1.7 焓——等压过程的自然变量 - 知乎](https://zhuanlan.zhihu.com/p/2062560709486158524)

## 第二章：热力学第二定律——过程的方向性（2.1 - 2.8）

> **核心问题**：什么样的热力学过程是可能的？自然界的方向性如何用数学语言精确表达？

### 2.1 热力学第二定律的物理直觉 [2.1 热力学第二定律的物理直觉 - 知乎](https://zhuanlan.zhihu.com/p/2062565318778681317)
- **核心任务**：建立第二定律的物理动机，理解第一定律的不足。
- 第一定律不限制过程的方向——能量守恒允许所有过程，但实际并非所有过程都能发生
- 举例：热量自发从高温物体传向低温物体（可以），反过来不行
- 举例：功可以完全转化为热（摩擦生热），热完全转化为功不行（热机有极限效率）
- 第二定律回答的问题：在能量守恒的前提下，哪些过程是可能的？方向如何？
- 自然过程的不可逆性——时间箭头的热力学起源

**推导目标**：
- 理解第一定律与第二定律的互补关系

### 2.2 热力学第二定律的两种经典表述 [2.2 热力学第二定律的两种经典表述 - 知乎](https://zhuanlan.zhihu.com/p/2062565575277254078)
- **核心任务**：建立第二定律的两种等价表述，理解它们的物理含义。
- 开尔文-普朗克表述：不可能从单一热源吸热完全转化为功而不产生其他影响（第二类永动机不可能）
- 克劳修斯表述：热量不可能自发地从低温物体传向高温物体而不产生其他影响
- 两种表述的等价性证明——用反证法：若一种表述被违反，则另一种也可被违反
- 热力学第二定律的本质：确定自然过程的方向性
- 第二定律与第一定律的关系：第一定律说能量守恒，第二定律说能量转换有方向

**推导目标**：
- 证明开尔文表述与克劳修斯表述的等价性

### 2.3 可逆过程与不可逆过程 [2.3 可逆过程与不可逆过程 - 知乎](https://zhuanlan.zhihu.com/p/2062565899379520347)
- **核心任务**：深刻理解可逆过程的理想化本质，掌握判定不可逆性的方法。
- 可逆过程的定义：系统经历一个过程后，系统和外界都能完全恢复到初始状态，不留下任何痕迹
- 可逆过程的条件：准静态（每一步都是平衡态）+ 无耗散（无摩擦、无非平衡过程）
- 典型的不可逆过程：
  - 摩擦耗散（机械能转化为内能）
  - 有限温差热传导（热量从高温到低温）
  - 自由膨胀（气体向真空膨胀）
  - 扩散（不同气体的混合）
- 不可逆性的根源：非平衡因素（梯度）驱动过程
- 可逆过程作为理论极限——实际过程的参照标准

**推导目标**：
- 分析自由膨胀为何不可逆
- 理解准静态与可逆的关系

### 2.4 卡诺循环与卡诺热机 [2.4 卡诺循环与卡诺热机 - 知乎](https://zhuanlan.zhihu.com/p/2062568361574371452)
- **核心任务**：建立卡诺循环的完整图像，计算其效率。
- 卡诺热机的四个步骤（两个等温 + 两个绝热）：
  1. 等温膨胀（高温 $T_h$，吸热 $Q_h$）
  2. 绝热膨胀（温度从 $T_h$ 降到 $T_c$）
  3. 等温压缩（低温 $T_c$，放热 $Q_c$）
  4. 绝热压缩（温度从 $T_c$ 升到 $T_h$）
- 卡诺循环的 $p$-$V$ 图和 $T$-$S$ 图
- 卡诺循环的效率推导：
  - $\eta = W/Q_h = (Q_h - Q_c)/Q_h$
  - 利用绝热过程关系推导 $Q_h/Q_c = T_h/T_c$
  - 最终 $\eta_C = 1 - T_c/T_h$

**推导目标**：
- 完整推导卡诺循环的效率
- 证明 $Q_h/Q_c = T_h/T_c$

### 2.5 卡诺定理 [2.5 卡诺定理 - 知乎](https://zhuanlan.zhihu.com/p/2062568398110958315)
- **核心任务**：证明所有可逆热机的效率上限，确立第二定律的定量表述。
- 卡诺定理的陈述：
  - 所有工作在两个给定热源之间的可逆热机效率相等（等于卡诺效率）
  - 不可逆热机的效率小于可逆热机
- 卡诺定理的证明（反证法）：
  - 假设存在效率更高的热机，用它驱动逆卡诺循环

**第一章（第一定律）** 确立能量守恒在热现象中的形式——热量和功的等价性，内能和焓作为状态函数，各种等值过程的计算。

**第二章（第二定律与熵）** 确立过程的方向性——引入熵作为状态函数，用熵增加原理判定过程方向，卡诺定理作为第二定律的定量表述。

**第三章（热力学函数与关系）** 建立四个热力学势的完整体系和麦克斯韦关系，为系统的热力学计算提供完整的数学工具，引入第三定律和吉布斯-亥姆霍兹方程。

**第四章（相变与相平衡）** 将热力学理论应用于物质相变——推导克拉佩龙方程、克劳修斯-克拉佩龙方程、吉布斯相律，建立相图的理论基础。

**第五章（多组分系统）** 将热力学推广到多组分系统——引入偏摩尔量和化学势，解决理想气体混合物和稀溶液的依数性问题。

**第六章（微观基础）** 建立宏观热力学与微观统计力学的联系——玻尔兹曼熵公式、熵增加原理的统计解释、时间箭头的物理本质。

**第七章（应用）** 展示热力学在热机、制冷、节流液化、热辐射和化学反应中的广泛应用。

**第八章（低温与量子）** 将热力学延伸到低温领域——超流、超导、玻色-爱因斯坦凝聚的定性概念。

整个课程的核心思想是：从两条经验定律（第一定律和第二定律）出发，通过严格的数学推理，建立一个能够描述和预测大量宏观热现象的完整理论体系。热力学不依赖微观模型，它的结论是普遍的、精确的——这正是宏观热力学的力量所在。
