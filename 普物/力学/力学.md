# 力学

> **课程定位**：大学物理五门基础课（力学、热学、光学、电磁学、原子物理）中的第一门。涵盖质点运动学、牛顿动力学、三大守恒律、万有引力与天体力学、刚体力学、弹性力学基础、流体力学基础、振动与波动、非线性振动与混沌入门。

### M.1 向量代数 [M.1 向量代数 - 知乎](https://zhuanlan.zhihu.com/p/2062422859478196964)

**核心任务**：系统回顾向量的基本运算，这是力学描述空间关系的语言。

**内容要点**：
- 向量的几何表示与分量表示：$\mathbf{r} = x\hat{i} + y\hat{j} + z\hat{k}$
- 向量的相等与自由向量概念
- 向量加减法的几何意义（平行四边形法则、三角形法则）
- 数乘向量与向量的线性组合
- 点乘（标量积）：$\mathbf{A}\cdot\mathbf{B} = AB\cos\theta$，物理意义——投影与功
- 点乘的交换律、分配律
- 点乘的分量形式：$A_xB_x + A_yB_y + A_zB_z$
- 叉乘（矢量积）：$\mathbf{A}\times\mathbf{B} = AB\sin\theta\,\hat{\mathbf{n}}$，物理意义——力矩与角动量
- 叉乘的反交换律、分配律，不满足结合律
- 叉乘的分量形式（行列式记法）
- 混合积的几何意义——平行六面体体积
- 混合积的分量形式与循环置换性质
- 矢量恒等式：$\mathbf{A}\times(\mathbf{B}\times\mathbf{C}) = \mathbf{B}(\mathbf{A}\cdot\mathbf{C}) - \mathbf{C}(\mathbf{A}\cdot\mathbf{B})$（BAC-CAB 恒等式）
- 单位向量与方向余弦
- 向量的模长与归一化

**推导目标**：
- 从基向量点乘关系 $\hat{i}\cdot\hat{i}=1$，$\hat{i}\cdot\hat{j}=0$ 导出点乘分量形式
- 从基向量叉乘关系 $\hat{i}\times\hat{j}=\hat{k}$ 等导出叉乘行列式形式
- 完整推导 BAC-CAB 恒等式（三重叉积展开）

## 第一章：质点运动学（1.1 - 1.7）


### 1.1 参考系与坐标系 [1.1 参考系与坐标系 - 知乎](https://zhuanlan.zhihu.com/p/2062426226329924255)

**核心任务**：建立“运动是相对的”这一核心物理直觉。

**内容要点**：
- 参考系的定义：描述运动所依据的物体（或物体群）
- 坐标系是参考系的数学化——给空间标尺
- 运动是相对的——绝对空间不存在（马赫原理的定性提及）
- 位置矢量 $\mathbf{r}(t)$：从原点到质点的有向线段
- 轨迹方程 $y(x)$ 与位置矢量 $\mathbf{r}(t)$ 的等价性
- 自由度的概念：描述一个质点需要几个独立坐标
- 位形空间的概念

**推导目标**：
- 明确位置矢量在三种坐标系中的表达形式


### 1.2 位移、速度与速率 [1.2 位移、速度与速率 - 知乎](https://zhuanlan.zhihu.com/p/2062426426146660523)

**核心任务**：从平均量过渡到瞬时量，引入微商运算。

**内容要点**：
- 位移 $\Delta \mathbf{r} = \mathbf{r}(t+\Delta t) - \mathbf{r}(t)$：路径无关
- 路程 $\Delta s$：轨迹的实际长度，路径依赖
- 平均速度 $\bar{\mathbf{v}} = \Delta\mathbf{r}/\Delta t$ 与瞬时速度 $\mathbf{v} = d\mathbf{r}/dt$
- 平均速率 $\bar{v} = \Delta s/\Delta t$ 与瞬时速率 $v = ds/dt$
- 速度的方向沿轨迹切线——曲线运动的理解关键
- 速度的分量表达：$v_x = \dot{x}, v_y = \dot{y}, v_z = \dot{z}$

**推导目标**：
- 从平均速度取极限 $\Delta t \to 0$ 导出瞬时速度
- 用速度定义推导匀速直线运动的位移公式


### 1.3 加速度 [1.3 加速度 - 知乎](https://zhuanlan.zhihu.com/p/2062426590609404822)

**核心任务**：理解加速度不仅改变速度大小，还改变速度方向。

**内容要点**：
- 平均加速度 $\bar{\mathbf{a}} = \Delta\mathbf{v}/\Delta t$ 与瞬时加速度 $\mathbf{a} = d\mathbf{v}/dt = d^2\mathbf{r}/dt^2$
- 加速度的分量表达：$a_x = \dot{v}_x = \ddot{x}$
- 匀加速直线运动的三个公式的微积分推导
- 重点推导：圆周运动的向心加速度 $a_n = v^2/R$
- 切向加速度 $a_t = dv/dt$（改变速率）与法向加速度 $a_n = v^2/\rho$（改变方向）
- 一般曲线运动中 $\rho$ 是曲率半径
- 匀速圆周运动的周期性

**推导目标**：
- 从 $v = v_0 + at$，$s = v_0t + \frac{1}{2}at^2$ 导出 $v^2 = v_0^2 + 2as$
- 从速度矢量的旋转推导向心加速度公式
- 从 $\mathbf{a} = d\mathbf{v}/dt$ 出发导出切向和法向加速度的分解形式


### 1.4 直角坐标系中的运动 [1.4 直角坐标系中的运动 - 知乎](https://zhuanlan.zhihu.com/p/2062426895199761307)

**核心任务**：将矢量方程转化为三个标量方程。

**内容要点**：
- $\mathbf{r}(t) = x(t)\hat{i} + y(t)\hat{j} + z(t)\hat{k}$
- $\mathbf{v}(t) = \dot{x}\hat{i} + \dot{y}\hat{j} + \dot{z}\hat{k}$
- $\mathbf{a}(t) = \ddot{x}\hat{i} + \ddot{y}\hat{j} + \ddot{z}\hat{k}$
- 直角坐标系中基向量方向固定，求导无额外项
- 两类问题：已知 $\mathbf{r}(t)$ 求 $\mathbf{v}, \mathbf{a}$（求导）；已知 $\mathbf{a}$ 和初始条件求 $\mathbf{v}, \mathbf{r}$（积分）

**推导目标**：
- 通过一个具体例子（如匀加速运动）完整展示两类问题的求解流程


### 1.5 抛体运动 [1.5 抛体运动 - 知乎](https://zhuanlan.zhihu.com/p/2062426988040680568)

**核心任务**：将微积分应用于真实世界。

**内容要点**：
- 初始条件设定：$\mathbf{r}(0) = 0$，$\mathbf{v}(0) = (v_0\cos\theta)\hat{i} + (v_0\sin\theta)\hat{j}$
- 动力学方程：$a_x = 0$，$a_y = -g$
- 积分得速度：$v_x = v_0\cos\theta$，$v_y = v_0\sin\theta - gt$
- 再积分得位置：$x = (v_0\cos\theta)t$，$y = (v_0\sin\theta)t - \frac{1}{2}gt^2$
- 消去参数 $t$ 得轨迹方程——抛物线
- 射程 $R = v_0^2\sin 2\theta/g$ 的推导
- 最大射程条件 $\theta = 45^\circ$
- 最大高度 $H = v_0^2\sin^2\theta/(2g)$ 的推导
- 空气阻力的定性影响——轨迹不再是对称抛物线

**推导目标**：
- 完整推导从初始条件到轨迹方程的全过程
- 推导射程和最大高度公式
- 分析射程随发射角变化的极值


### 1.6 自然坐标系 [1.6 自然坐标系 - 知乎](https://zhuanlan.zhihu.com/p/2062427137345429525)

**核心任务**：处理沿已知轨迹的运动。

**内容要点**：
- 自然坐标 $s(t)$：沿轨迹从某参考点量取的路程
- 切向单位向量 $\hat{\tau}$：沿轨迹前进方向，$\hat{\tau} = d\mathbf{r}/ds$
- 法向单位向量 $\hat{n}$：指向轨迹凹侧（曲率中心方向）
- 速度的分解：$\mathbf{v} = v\hat{\tau}$，$v = \dot{s}$
- 加速度的完整推导：$\mathbf{a} = \dot{v}\hat{\tau} + (v^2/\rho)\hat{n}$
- 圆轨道特例：$\rho = R$，$a_n = v^2/R$
- 过山车在最高点和最低点的受力分析

**推导目标**：
- 推导 $d\hat{\tau}/dt$ 的表达式
- 完整导出自然坐标系中的加速度公式


### 1.7 极坐标系中的运动 [1.7 极坐标系中的运动 - 知乎](https://zhuanlan.zhihu.com/p/2062427447791031388)

**核心任务**：处理绕固定点运动的问题。

**内容要点**：
- 极坐标 $(r,\theta)$：基向量 $\hat{r}$（径向向外）和 $\hat{\theta}$（角度增加方向）
- 位置矢量：$\mathbf{r} = r\hat{r}$
- 基向量随时间变化：$\dot{\hat{r}} = \dot{\theta}\hat{\theta}$，$\dot{\hat{\theta}} = -\dot{\theta}\hat{r}$
- 速度：$\mathbf{v} = \dot{r}\hat{r} + r\dot{\theta}\hat{\theta}$
- 加速度完整推导：$\mathbf{a} = (\ddot{r} - r\dot{\theta}^2)\hat{r} + (r\ddot{\theta} + 2\dot{r}\dot{\theta})\hat{\theta}$
- 径向加速度 $a_r = \ddot{r} - r\dot{\theta}^2$
- 横向加速度 $a_\theta = r\ddot{\theta} + 2\dot{r}\dot{\theta}$（科里奥利加速度的来源）
- 圆周运动时 $r$ 为常数，回到 $a_r = -r\dot{\theta}^2$，$a_\theta = r\ddot{\theta}$

**推导目标**：
- 从位置矢量出发完整推导极坐标中的速度和加速度表达式
- 展示所有求导步骤，不省略基向量导数项

### 2.1 牛顿三定律 [2.1 牛顿三定律 - 知乎](https://zhuanlan.zhihu.com/p/2062427910322102801)

**核心任务**：深刻理解三个定律各自的内容、适用范围和常见误解。

**内容要点**：
- 第一定律（惯性定律）：不受外力时，物体保持静止或匀速直线运动
- 惯性参考系的定义（第一定律定义了惯性系）
- 第二定律：$\sum\mathbf{F} = d\mathbf{p}/dt = m\mathbf{a}$（质量恒定）
- 矢量方程，分量形式：$\sum F_x = ma_x$，$\sum F_y = ma_y$，$\sum F_z = ma_z$
- 瞬时性：力和加速度同时存在、同时变化
- 因果性：力是因，加速度是果
- 第三定律：$\mathbf{F}_{12} = -\mathbf{F}_{21}$
- 作用力和反作用力作用在不同物体上，不能抵消
- 与平衡力的区别（平衡力作用在同一物体上）
- 牛顿力学的适用范围：宏观、低速、弱引力

**推导目标**：
- 从第二定律推导动量定理的微分形式 $\mathbf{F} = d\mathbf{p}/dt$


### 2.2 常见的力——重力、弹力、摩擦力、张力 [2.2 常见的力——重力、弹力、摩擦力、张力 - 知乎](https://zhuanlan.zhihu.com/p/2062428062361428767)

## 逻辑关系总览

整个力学课程的体系有清晰的递进层次。

**第零章**是数学工具箱，为后续所有章节提供语言。

**第一章（运动学）** 回答“怎么动”——用位置、速度、加速度建立运动的几何描述。

**第二章（动力学）** 回答“为什么这样动”——引入力、质量、牛顿定律作为运动的原因。

**第三章（动量与能量）** 回答“有没有更普适的规律”——引入守恒律，展示对称性如何决定守恒量。

**第四章（角动量与万有引力）** 是守恒律在旋转和天体运动中的应用，展示从开普勒定律到万有引力定律的完整推导。

**第五章（刚体力学）** 将力学推广到非质点物体，引入转动惯量和角动量。

**第六章（弹性力学）** 和 **第七章（流体力学）** 构成连续介质力学的两翼，展示力学如何处理“无限多质点”的系统。

**第八章（振动与波动）** 展示局域振动如何在空间中传播，是连接力学与光学、声学的桥梁。

**第九章（非线性振动与混沌）** 是力学的现代视角——即使牛顿方程是确定性的，世界在长时间尺度上依然不可预测。
