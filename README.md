# 物理学课程笔记

一个持续整理中的中文物理学习笔记库，覆盖从普通物理到量子场论、格点 QCD 等主题。笔记以课程和章节为单位组织，侧重概念梳理、理论推导与知识体系的建立。

## 适用读者

这些笔记主要面向物理学专业高年级本科生和研究生，也适合希望系统复习理论物理的读者。基础部分通常需要微积分、线性代数和常微分方程知识；进阶部分默认读者具备相应的经典力学、电动力学、统计物理或量子力学基础。

## 建议学习顺序

1. 从普通物理建立对力学、热学、电磁学、光学和原子物理的整体认识。
2. 进入四大力学，系统学习分析力学、电动力学、统计物理和量子力学。
3. 在上述基础上学习广义相对论或量子场论，两者可以根据兴趣分别展开。
4. 完成量子场论基础后，再阅读有效场论、重整化群、拓扑量子场论、量子色动力学和格点 QCD 等专题。

这一路线仅用于展示课程之间的大致依赖关系，不要求严格按顺序阅读。
