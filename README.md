# 物理学课程笔记

一个持续整理中的中文物理学习笔记库，覆盖从普通物理到量子场论、格点 QCD 等主题。笔记以 Markdown 编写，公式使用 LaTeX 语法，可直接在 GitHub 阅读，也适合用 [Obsidian](https://obsidian.md/) 打开和浏览。

## 内容导航

| 目录 | 主要内容 |
| --- | --- |
| [四大力学](./四大力学) | 分析力学、电动力学、统计物理、量子力学 |
| [普物](./普物) | 力学、热力学、电磁学、光学、原子物理 |
| [广义相对论](./广义相对论) | 广义相对论相关课程笔记 |
| [量子场论](./量子场论) | 量子场论基础与进阶主题 |
| [量子场论专题](./量子场论专题) | 有效场论与重整化群、拓扑量子场论、拓扑粒子、量子色动力学 |
| [格点量子色动力学](./格点量子色动力学) | 格点规范场论、数值模拟、强子物理、有限温度 QCD 等 |

各课程目录通常包含目录页和按章节组织的笔记。建议从各子目录中的“目录”或同名 Markdown 文件开始阅读。

## 使用方式

### 在 GitHub 阅读

直接通过上方目录进入相应主题。GitHub 会渲染大部分 Markdown 内容和 LaTeX 公式。

### 在本地阅读

克隆仓库后，用任意 Markdown 编辑器打开即可；使用 Obsidian 时，可将仓库根目录作为一个 Vault 打开。

```bash
git clone https://github.com/meteor1937/phy-notes.git
```
