## 5.7 波导与谐振腔中的模式分析

前面几节我们讨论了电磁波在无界空间中的传播——平面波在真空中以光速 $c$ 传播，在介质中速度减慢，在导体表面反射和折射。这些讨论有一个共同的前提：空间是无限大的，波可以自由地向任意方向传播，没有任何边界约束。

但在无数实际应用中，我们需要将电磁波限制在有限的空间内传播。光纤将光约束在微米级的纤芯中跨越大洋；金属波导将微波从发射机导向天线；谐振腔将电磁能量储存在封闭的金属盒中，形成稳定的驻波模式。这些结构共同的工作原理是：利用导体表面或介质界面的反射，将电磁波约束在有限区域内，使得只有满足特定条件的“模式”能够存在和传播。

本节的任务是：从麦克斯韦方程组和理想导体边界条件出发，系统推导矩形金属波导中的导波模式（TE 模和 TM 模），引入截止频率的概念，然后将波导在纵向封闭为谐振腔，分析其三维驻波模式和离散共振频率。


### 一、为什么需要波导——从双线传输到金属管

在低频电路中，能量通过双线传输线或同轴电缆传输。但当频率升高到微波波段（几 GHz 到几十 GHz），普通的双线传输线面临两个致命问题。一是辐射损耗——导线像天线一样向空间辐射电磁能量，频率越高，辐射越强，传输效率急剧下降。二是趋肤效应——5.4 节讨论过，高频电流只集中在导体表面极薄的层内，导线的有效电阻随频率升高而急剧增大，欧姆损耗变得不可接受。

金属波导解决了这两个问题。它是一个中空的金属管，截面通常为矩形或圆形。电磁波在管内传播，被金属壁完全约束——能量无法泄漏到外部空间，辐射损耗被消除。金属壁的趋肤效应使电流只在管内表面流动，欧姆损耗远小于开放导线（电流分布在更宽阔的表面上）。波导成为微波频段最核心的传输结构，从雷达系统到卫星通信，从粒子加速器到微波炉，波导无处不在。


### 二、理想导体边界条件与波导中的一般设定

在分析波导之前，我们需要一个金属表面的边界条件。对于**理想导体**（电导率 $\sigma \to \infty$），5.4 节告诉我们趋肤深度趋于零——电磁场完全无法进入导体内部。在导体表面上，电场必须满足：切向电场为零。如果不是零，表面切向电场会驱动无穷大的传导电流，这在物理上是不可能的（至少在没有外加电源时）。因此：

$$
\hat{\mathbf{n}}\times\mathbf{E} = 0 \quad (\text{在理想导体表面})
$$

对于磁场，由于导体内部 $\mathbf{B}=0$，而磁场的法向分量在界面上连续（$\nabla\cdot\mathbf{B}=0$ 的积分形式），因此表面上磁场的法向分量为零：

$$
\hat{\mathbf{n}}\cdot\mathbf{B} = 0 \quad (\text{在理想导体表面})
$$

这两个边界条件是整个波导和腔模分析的基石。

考虑一个无限长的直波导，截面为矩形（$x$ 方向宽 $a$，$y$ 方向高 $b$），管壁是理想导体。波导内部是真空（或空气，$\epsilon_0, \mu_0$）。电磁波沿 $z$ 方向传播，但在 $x$ 和 $y$ 方向受导体壁约束，形成驻波。我们寻找沿 $z$ 方向传播的单色波解，电场和磁场的形式为：

$$
\mathbf{E}(x,y,z,t) = \mathbf{E}_0(x,y) e^{i(k_g z - \omega t)}
$$
$$
\mathbf{B}(x,y,z,t) = \mathbf{B}_0(x,y) e^{i(k_g z - \omega t)}
$$

其中 $k_g$ 是**导波波长对应的波数**（guide wavenumber）——注意它不等于自由空间的波数 $k = \omega/c$。这正是因为横向约束改变了波的色散关系。


### 三、纵向场分量的分离与 TE/TM 分类

波导内部无源（$\rho=0, \mathbf{J}=0$），电场和磁场各自满足亥姆霍兹方程：

$$
\nabla^2\mathbf{E} + k^2\mathbf{E} = 0, \quad \nabla^2\mathbf{B} + k^2\mathbf{B} = 0
$$

其中 $k = \omega/c$ 是同一频率下真空中的波数。将拉普拉斯算符分解为横向（$xy$）和纵向（$z$）两部分：$\nabla^2 = \nabla_t^2 + \partial^2/\partial z^2$。由于场对 $z$ 的依赖是 $e^{ik_g z}$，$\partial^2/\partial z^2 \to -k_g^2$。因此亥姆霍兹方程化为：

$$
\nabla_t^2\mathbf{E} + (k^2 - k_g^2)\mathbf{E} = 0
$$

定义**截止波数** $k_c$：

$$
\boxed{k_c^2 \equiv k^2 - k_g^2 = \frac{\omega^2}{c^2} - k_g^2}
$$

这样，亥姆霍兹方程在横截面上具有标准形式：$\nabla_t^2\mathbf{E} + k_c^2\mathbf{E} = 0$。$k_c$ 由横截面的边界条件决定——它是本征值问题的本征值。

波导中的电磁场模式可以分为两类。**横电波（TE 波）**：电场纯粹是横向的，$E_z = 0$，但磁场有纵向分量 $B_z \neq 0$。**横磁波（TM 波）**：磁场纯粹是横向的，$B_z = 0$，但电场有纵向分量 $E_z \neq 0$。为什么不能同时 $E_z = B_z = 0$（真正的 TEM 波）？因为在空心金属波导中，横向电场线必须起始和终止于导体壁上的电荷——如果 $E_z = 0$ 且导体壁上的纵向电场也为零（边界条件），横向电场可以由壁上的电荷产生。但横向磁场线是闭合的——它们必须环绕纵向电流或位移电流。如果没有纵向电场和纵向磁场，位移电流也只能是横向的，无法产生环绕纵向的横向磁场。数学上可以严格证明，空心单导体波导不支持 TEM 模式。同轴电缆（双导体）支持 TEM 波，但矩形波导只能支持 TE 和 TM 模式。


### 四、矩形波导中的 TE 模式

对于 TE 波，$E_z = 0$，我们从 $B_z$ 出发求解。$B_z$ 满足亥姆霍兹方程 $\nabla_t^2 B_z + k_c^2 B_z = 0$。这是横截面上的标量亥姆霍兹方程，需要在矩形边界 $x \in [0, a], y \in [0, b]$ 上施加边界条件。

边界条件是：在理想导体表面，电场切向分量为零。对于 TE 波，$E_z=0$ 自动满足，但 $E_x$ 和 $E_y$ 必须在边界上为零。$E_x$ 和 $E_y$ 与 $B_z$ 的关系由麦克斯韦方程组的旋度方程导出。将安培-麦克斯韦定律和法拉第定律写成分量形式，在 $e^{i(k_g z - \omega t)}$ 的约定下，横向电场分量可以用 $B_z$ 表达。在边界 $x=0$ 和 $x=a$ 上，法向方向是 $\hat{\mathbf{x}}$，切向电场包括 $E_y$。$E_y$ 正比于 $\partial B_z/\partial x$（具体系数来自旋度方程）。要求 $E_y=0$ 意味着 $\partial B_z/\partial x = 0$ 在 $x=0$ 和 $x=a$ 处。类似地，在 $y=0$ 和 $y=b$ 处，$E_x \propto \partial B_z/\partial y = 0$。

因此 $B_z$ 满足的边界条件是：**在边界上法向导数为零**（诺伊曼条件）：

$$
\frac{\partial B_z}{\partial x}\bigg|_{x=0,a} = 0, \quad \frac{\partial B_z}{\partial y}\bigg|_{y=0,b} = 0
$$

求解 $\nabla_t^2 B_z + k_c^2 B_z = 0$，分离变量 $B_z(x,y) = X(x)Y(y)$。代入得：

$$
\frac{X''}{X} + \frac{Y''}{Y} + k_c^2 = 0
$$

每项分别等于常数：$X''/X = -k_x^2$，$Y''/Y = -k_y^2$，且 $k_x^2 + k_y^2 = k_c^2$。诺伊曼边界条件要求 $X'(0) = X'(a) = 0$，因此 $X(x) \propto \cos(k_x x)$，且 $k_x = m\pi/a$（$m = 0,1,2,\ldots$）。同理 $Y(y) \propto \cos(k_y y)$，$k_y = n\pi/b$（$n = 0,1,2,\ldots$）。$m$ 和 $n$ 不能同时为零（否则 $B_z$ 为常数，所有横向场消失）。因此：

$$
\boxed{B_z(x,y) = B_0 \cos\left(\frac{m\pi x}{a}\right)\cos\left(\frac{n\pi y}{b}\right)}
$$

对应的截止波数 $k_c$ 为：

$$
\boxed{k_{c,mn} = \pi\sqrt{\left(\frac{m}{a}\right)^2 + \left(\frac{n}{b}\right)^2}}
$$

这就是 ** $\text{TE}_{mn}$ 模式**的场分布。知道了 $B_z$，可以通过旋度方程求出所有其他分量（$E_x, E_y, B_x, B_y$）。这些横向场与 $B_z$ 成正比，并具有与 $B_z$ 相同的横向节点结构。


### 五、矩形波导中的 TM 模式

对于 TM 波，$B_z = 0$，我们从 $E_z$ 出发。$E_z$ 满足同样的亥姆霍兹方程。边界条件是：在导体表面，电场的切向分量为零。$E_z$ 自身就是切向分量（它平行于 $z$ 方向，而导体壁的法线在 $x$ 或 $y$ 方向），因此 $E_z$ 在边界上直接为零：

$$
E_z(0,y) = E_z(a,y) = 0, \quad E_z(x,0) = E_z(x,b) = 0
$$

这是狄利克雷条件。分离变量 $E_z = X(x)Y(y)$，边界条件要求 $X(0) = X(a) = 0$，因此 $X(x) \propto \sin(k_x x)$，$k_x = m\pi/a$，** $m = 1,2,3,\ldots$ **（$m=0$ 给出零解）。同理 $k_y = n\pi/b$，** $n = 1,2,3,\ldots$ **。因此：

$$
\boxed{E_z(x,y) = E_0 \sin\left(\frac{m\pi x}{a}\right)\sin\left(\frac{n\pi y}{b}\right)}
$$

截止波数 $k_{c,mn}$ 与 TE 模式形式上相同（同样的 $k_x^2 + k_y^2 = k_c^2$），但 $m,n$ 的取值范围不同（TM 模中 $m,n \ge 1$，TE 模中 $m,n \ge 0$ 但不能同时为零）。这就是 ** $\text{TM}_{mn}$ 模式**。


### 六、截止频率与导波的色散关系

$k_c$ 是波导的本征值，由截面几何形状和模式序数 $m,n$ 决定。由 $k_c^2 = k^2 - k_g^2$，代入 $k = \omega/c$：

$$
k_g = \sqrt{\frac{\omega^2}{c^2} - k_c^2}
$$

$k_g$ 必须为实数，波才能沿 $z$ 方向传播（否则 $e^{ik_g z}$ 变为指数衰减）。因此，对于一个给定模式 $\text{TE}_{mn}$ 或 $\text{TM}_{mn}$，只有频率满足 $\omega \ge c k_c$ 时，波才能无衰减地传播。定义**截止频率**：

$$
\boxed{\omega_{c,mn} \equiv c k_{c,mn} = c\pi\sqrt{\left(\frac{m}{a}\right)^2 + \left(\frac{n}{b}\right)^2}}
$$

或写为截止波长：$\lambda_{c,mn} = 2\pi/k_{c,mn}$。当 $\omega < \omega_c$ 时，$k_g$ 为虚数，该模式不能在波导中传播——它是**截止的**，场在纵向指数衰减（倏逝模）。当 $\omega > \omega_c$ 时，该模式是**传播的**。波导就像一个高通滤波器——只有频率足够高的信号才能通过。

最低阶的模式（截止频率最低）通常是 $\text{TE}_{10}$（$m=1, n=0$，假设 $a > b$）。其截止频率为 $\omega_{c,10} = \pi c/a$，截止波长为 $\lambda_{c,10} = 2a$ ——恰好是波导宽边尺寸的两倍。这意味着：**波导只能传播波长小于两倍宽边尺寸的电磁波**。这就是为什么波导尺寸总是与工作波长可比——对于 $10\,\text{GHz}$ 的雷达波（$\lambda = 3\,\text{cm}$），波导宽边约为几厘米。对于可见光（$\lambda \sim 500\,\text{nm}$），要制作相应的金属波导需要纳米级的加工精度——不现实，但介质波导（光纤）可以在光学频段工作。

波导中的导波波长 $\lambda_g = 2\pi/k_g$ **大于**同一频率下的自由空间波长 $\lambda = 2\pi/k = 2\pi c/\omega$。导波波数 $k_g = \sqrt{k^2 - k_c^2} < k$，因此相速度 $v_p = \omega/k_g$ **大于**光速 $c$。这并不违反相对论——相速度是等相位面的移动速度，不携带信息和能量。能量以**群速度**传播：$v_g = d\omega/dk_g = c\sqrt{1 - (\omega_c/\omega)^2} < c$。波导中的色散关系 $\omega^2 = c^2(k_g^2 + k_c^2)$ 类似于相对论粒子的能量-动量关系（$E^2 = p^2c^2 + m^2c^4$），其中 $k_c$ 扮演了“截止质量”的角色。


### 七、谐振腔——将波导在纵向封闭

将一段长度为 $d$ 的波导两端用理想导体板封闭，电磁波在纵向也被反射，形成三维驻波——这就是**谐振腔**。波在两端之间来回反射，在特定的频率下所有反射波同相叠加，形成共振。谐振腔在微波振荡器、滤波器和粒子加速器中广泛使用。

在谐振腔中，电磁场不仅在横向形成驻波，在纵向也形成驻波。纵向边界条件（在 $z=0$ 和 $z=d$ 处的理想导体壁）要求在该处切向电场为零。对于 $\text{TE}_{mn}$ 模式，横向电场在 $z=0$ 和 $z=d$ 处必须为零，这要求纵向波数 $k_g$ 满足驻波条件：$k_g = l\pi/d$，$l = 1,2,3,\ldots$（对于 TE 模，$l=0$ 给出的解不满足边界条件或导致场恒为零）。对于 $\text{TM}_{mn}$ 模式，$l$ 可以为零（此时场在纵向均匀，但需检验边界条件）。

由色散关系 $k^2 = k_c^2 + k_g^2$，代入 $k = \omega/c$，$k_c = k_{c,mn}$，$k_g = l\pi/d$，得到谐振腔的**共振频率**：

$$
\boxed{\omega_{mnl} = c\pi\sqrt{\left(\frac{m}{a}\right)^2 + \left(\frac{n}{b}\right)^2 + \left(\frac{l}{d}\right)^2}}
$$

这是谐振腔中 $\text{TE}_{mnl}$ 和 $\text{TM}_{mnl}$ 模式的共振频率。注意它与波导截止频率的区别：在波导中，$\omega$ 可以连续变化（只要大于 $\omega_c$）；在谐振腔中，只有满足三维驻波条件的**离散频率**才能存在—— $\omega_{mnl}$ 是分立的频谱。

谐振腔的品质因数 $Q$（品质因子）描述了腔储存能量的效率：$Q = \omega_0 \times (\text{储存能量}) / (\text{损耗功率})$。对于超导谐振腔，$Q$ 可达 $10^{10}$ 以上，是粒子加速器和量子比特读出电路的核心元件。


### 八、从波导到光纤——介质波导的概念延伸

金属波导在微波频段广泛应用，但在光频段不可行——金属在光频不再是理想导体，损耗巨大，且趋肤深度极小，表面粗糙度要求极高。光频段的“波导”是光纤——一种介质波导。光纤利用全内反射（5.6 节）将光约束在纤芯中：纤芯折射率 $n_1$ 略大于包层折射率 $n_2$，当光以大于临界角的角度入射到纤芯-包层界面时，发生全内反射，光被完全约束在纤芯中传播。

与金属波导不同，介质波导可以支持真正的导模（guided modes），这些模式的场在包层中指数衰减（类似于全内反射的倏逝波），在纤芯中振荡。介质波导没有严格截止的“全反射”模式——它总是支持至少一个基模（$\text{HE}_{11}$ 模），在弱导近似下退化为线偏振 $\text{LP}_{01}$ 模。单模光纤（芯径约 $8\text{-}10\,\mu\text{m}$）只支持这一个模式，是现代长距离光通信的核心。

波导和谐振腔的理论深刻改变了我们对电磁波传播的理解。在无界空间中，电磁波可以以任何频率和任何方向自由传播。但在有边界的结构中，边界条件将连续谱“压缩”为分立的模式——每个模式有特定的场分布（横向节点结构）和特定的传播特性（色散关系、截止频率）。这种“模式化”是波动系统在有限空间中必然出现的普遍现象——从量子力学中的无限深势阱到固体物理中的声子态密度，从激光谐振腔的纵模到光纤通信的波分复用，模式分析的思想贯穿整个现代物理学和工程学。
