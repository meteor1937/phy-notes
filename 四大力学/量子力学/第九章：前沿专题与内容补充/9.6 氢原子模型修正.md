在前一篇中，我们求解了氢原子的薛定谔方程，得到了能级 $E_n = -13.6\,\text{eV}/n^2$ 和波函数 $\psi_{nlm}$。这个“裸”氢原子模型抓住了问题的主要部分，但真实的氢原子光谱远比这复杂。高分辨光谱实验揭示了能级的进一步分裂——精细结构、超精细结构，以及兰姆移位。这些修正一个比一个小，但每一个都打开了一扇通往新物理的窗口：相对论、电子自旋、量子电动力学，乃至原子核的内部结构。

这一次，我们以系统的方式，逐层引入氢原子模型的各种修正，并用微扰论精确计算每一层修正带来的能级移动。

* * *

### 一、基础模型的局限与修正路线图

薛定谔方程给出的氢原子能级：

$$E_n = -\frac{R_y}{n^2}, \quad R_y = \frac{m_e e^4}{2(4\pi\epsilon_0)^2 \hbar^2} = 13.6\,\text{eV}$$

只依赖于主量子数 $n$，对 $l$ 和 $m$ 高度简并。但实验告诉我们：

1.  **$l$ 简并被解除**：$2s$ 和 $2p$ 的能级并不完全重合（忽略兰姆移位，它们仍有微小的分裂）。
2.  **$j$ 相关的分裂**：考虑自旋后，每条谱线实际上由间距很小的多条谱线组成（精细结构）。
3.  **更细微的分裂**：即使在总角动量 $j$ 相同的态之间，也存在极小的能级差（兰姆移位）。
4.  **与核自旋的耦合**：电子与核磁矩的相互作用导致超精细分裂（如 21 cm 线）。

修正哈密顿量可以写为：

$$\hat{H} = \hat{H}_0 + \hat{H}_{\text{rel}} + \hat{H}_{\text{SO}} + \hat{H}_{\text{Darwin}} + \hat{H}_{\text{hf}} + \hat{H}_{\text{QED}} + \cdots$$

其中：

-   $\hat{H}_0$：非相对论薛定谔哈密顿量（已解）
-   $\hat{H}_{\text{rel}}$：相对论动能修正（$p^4$ 项）
-   $\hat{H}_{\text{SO}}$：自旋-轨道耦合
-   $\hat{H}_{\text{Darwin}}$：达尔文项（电子的相对论颤动）
-   $\hat{H}_{\text{hf}}$：超精细结构（电子与核自旋耦合）
-   $\hat{H}_{\text{QED}}$：量子电动力学修正（兰姆移位）

我们逐项处理这些修正。

* * *

### 二、动能修正：相对论 $p^4$ 项

### 2.1 物理来源

薛定谔方程使用的是非相对论动能 $T = p^2/(2m)$。根据狭义相对论，动能的正确表达式为：

$$T = \sqrt{p^2 c^2 + m^2 c^4} - m c^2$$

在低能极限下，可以按 $p^2/(m^2 c^2)$ 展开：

$$T = \frac{p^2}{2m} - \frac{p^4}{8 m^3 c^2} + \cdots$$

第一项是非相对论动能，第二项是**最低阶的相对论修正**。因此，相对论动能修正哈密顿量为：

$$\hat{H}_{\text{rel}} = -\frac{\hat{p}^4}{8 m_e^3 c^2}$$

### 2.2 一阶微扰计算

我们用未微扰的氢原子波函数 $\psi_{nlm}$ 计算一阶能量修正：

$$\Delta E_{\text{rel}}^{(1)} = \langle \psi_{nlm} | \hat{H}_{\text{rel}} | \psi_{nlm} \rangle = -\frac{1}{8 m_e^3 c^2} \langle \psi_{nlm} | \hat{p}^4 | \psi_{nlm} \rangle$$

直接计算 $\hat{p}^4$ 的期望值较为复杂，但我们可以巧妙地利用未微扰的薛定谔方程。对于氢原子，有：

$$\hat{H}_0 = \frac{\hat{p}^2}{2m_e} - \frac{k}{r}, \quad k = \frac{e^2}{4\pi\epsilon_0}$$

因此 $\hat{p}^2 = 2m_e (\hat{H}_0 + k/r)$。进而：

$$\hat{p}^4 = 4 m_e^2 \left( \hat{H}_0 + \frac{k}{r} \right)^2$$

在 $\psi_{nlm}$ 态下，$\hat{H}_0$ 作用给出 $E_n$。因此：

$$\langle \hat{p}^4 \rangle = 4 m_e^2 \left\langle \left( E_n + \frac{k}{r} \right)^2 \right\rangle = 4 m_e^2 \left( E_n^2 + 2k E_n \left\langle \frac{1}{r} \right\rangle + k^2 \left\langle \frac{1}{r^2} \right\rangle \right)$$

现在需要氢原子的期望值 $\langle 1/r \rangle$ 和 $\langle 1/r^2 \rangle$。这些可以用 Kramers 公式或直接积分得到。回忆标准结果（在上一篇中提到）：

$$\left\langle \frac{1}{r} \right\rangle_{nlm} = \frac{1}{n^2 a_0}$$

$$\left\langle \frac{1}{r^2} \right\rangle_{nlm} = \frac{1}{(l + 1/2) n^3 a_0^2}$$

其中 $a_0 = \hbar^2/(m_e k)$ 是玻尔半径。注意 $E_n = -k/(2 n^2 a_0) = -R_y / n^2$。

代入这些：

$$\begin{aligned} \langle \hat{p}^4 \rangle &= 4 m_e^2 \left( E_n^2 + 2k E_n \frac{1}{n^2 a_0} + k^2 \frac{1}{(l+1/2)n^3 a_0^2} \right) \\ &= 4 m_e^2 E_n^2 \left( 1 + \frac{2k}{E_n n^2 a_0} + \frac{k^2}{E_n^2 (l+1/2) n^3 a_0^2} \right) \end{aligned}$$

利用 $E_n = -k/(2n^2 a_0)$，则 $k/(E_n n^2 a_0) = -2$，$k^2/(E_n^2 n^3 a_0^2) = 4n$。因此：

$$\langle \hat{p}^4 \rangle = 4 m_e^2 E_n^2 \left( 1 - 4 + \frac{4n}{l+1/2} \right) = 4 m_e^2 E_n^2 \left( \frac{4n}{l+1/2} - 3 \right)$$

所以：

$$\Delta E_{\text{rel}}^{(1)} = -\frac{1}{8 m_e^3 c^2} \cdot 4 m_e^2 E_n^2 \left( \frac{4n}{l+1/2} - 3 \right)$$

化简，并用 $E_n = -R_y/n^2$，$R_y = m_e e^4/[2(4\pi\epsilon_0)^2\hbar^2] = m_e k^2/(2\hbar^2)$，以及精细结构常数 $\alpha = e^2/(4\pi\epsilon_0 \hbar c) = k/(\hbar c) \approx 1/137$：

$$\boxed{\Delta E_{\text{rel}}^{(1)} = -E_n \frac{\alpha^2}{n^2} \left( \frac{n}{l+1/2} - \frac{3}{4} \right)}$$

或者写作：

$$\Delta E_{\text{rel}}^{(1)} = -\frac{(E_n)^2}{2 m_e c^2} \left( \frac{4n}{l+1/2} - 3 \right)$$

这给出的是相对论动能修正对能级的贡献。对于氢原子基态 $n=1, l=0$：

$$\Delta E_{\text{rel}}^{(1)}(1s) = -E_1 \alpha^2 \left( \frac{1}{1/2} - \frac{3}{4} \right) = -E_1 \alpha^2 \left( 2 - \frac{3}{4} \right) = -\frac{5}{4} E_1 \alpha^2$$

数值上，$E_1 = -13.6$ eV，$\alpha^2 \approx 5.33 \times 10^{-5}$，因此 $\Delta E_{\text{rel}}^{(1)}(1s) \approx -9.1 \times 10^{-4}$ eV。这是一个很小的修正（约为基态能量的 $6.7 \times 10^{-5}$），但高分辨光谱完全可以分辨。

### 三、自旋-轨道耦合

### 3.1 物理图像

电子绕核运动时，在电子自身的静止系中，带正电的原子核围绕电子运动，产生一个磁场。这个磁场与电子的自旋磁矩相互作用，产生能级分裂。这就是**自旋-轨道耦合**。

从相对论量子力学（狄拉克方程）可以直接导出，自旋-轨道耦合哈密顿量为：

$$\hat{H}_{\text{SO}} = \frac{1}{2 m_e^2 c^2} \frac{1}{r} \frac{dV}{dr} \hat{\mathbf{L}} \cdot \hat{\mathbf{S}}$$

对于库仑势 $V(r) = -k/r$，$dV/dr = k/r^2$。因此：

$$\hat{H}_{\text{SO}} = \frac{k}{2 m_e^2 c^2} \frac{1}{r^3} \hat{\mathbf{L}} \cdot \hat{\mathbf{S}}$$

### 3.2 耦合基底的选择

$\hat{H}_{\text{SO}}$ 中包含 $\hat{\mathbf{L}} \cdot \hat{\mathbf{S}}$。在非耦合基底 $|l, m_l\rangle \otimes |s, m_s\rangle$ 下，这个算符不是对角的。我们需要转到耦合基底。

定义总角动量：

$$\hat{\mathbf{J}} = \hat{\mathbf{L}} + \hat{\mathbf{S}}$$

则 $\hat{J}^2, \hat{J}_z, \hat{L}^2, \hat{S}^2$ 构成对易算符完备集。耦合态记为 $|n, l, s, j, m_j\rangle$。$s=1/2$ 是固定的。

在耦合基底中：

$$\hat{\mathbf{L}} \cdot \hat{\mathbf{S}} = \frac{1}{2} (\hat{J}^2 - \hat{L}^2 - \hat{S}^2)$$

其本征值为：

$$\langle \hat{\mathbf{L}} \cdot \hat{\mathbf{S}} \rangle = \frac{\hbar^2}{2} \left[ j(j+1) - l(l+1) - s(s+1) \right]$$

$s=1/2$，因此：

$$\langle \hat{\mathbf{L}} \cdot \hat{\mathbf{S}} \rangle = \frac{\hbar^2}{2} \left[ j(j+1) - l(l+1) - \frac{3}{4} \right]$$

对于 $l \neq 0$，$j = l \pm 1/2$。

### 3.3 能量修正

一阶微扰修正为：

$$\Delta E_{\text{SO}}^{(1)} = \frac{k}{2 m_e^2 c^2} \left\langle \frac{1}{r^3} \right\rangle \langle \hat{\mathbf{L}} \cdot \hat{\mathbf{S}} \rangle$$

氢原子的期望值 $\langle 1/r^3 \rangle$：

$$\left\langle \frac{1}{r^3} \right\rangle_{nl} = \frac{1}{a_0^3 n^3 l(l+1/2)(l+1)}$$

（注意这个公式对 $l=0$ 发散——我们稍后处理 $s$ 态的情况）。

代入并利用 $\alpha = k/(\hbar c)$，$a_0 = \hbar^2/(m_e k)$，经过代数整理，得到：

$$\boxed{\Delta E_{\text{SO}}^{(1)} = -E_n \frac{\alpha^2}{2n} \cdot \frac{j(j+1) - l(l+1) - 3/4}{l(l+1/2)(l+1)}}$$

对于 $j = l + 1/2$：

$$j(j+1) - l(l+1) - \frac{3}{4} = \left(l+\frac{1}{2}\right)\left(l+\frac{3}{2}\right) - l(l+1) - \frac{3}{4} = l$$

对于 $j = l - 1/2$（$l \ge 1$）：

$$j(j+1) - l(l+1) - \frac{3}{4} = \left(l-\frac{1}{2}\right)\left(l+\frac{1}{2}\right) - l(l+1) - \frac{3}{4} = -l - 1$$

因此：

$$\boxed{\Delta E_{\text{SO}}^{(1)} = \begin{cases} -E_n \dfrac{\alpha^2}{2n} \cdot \dfrac{1}{(l+1/2)(l+1)}, & j = l + 1/2 \\[1.5em] +E_n \dfrac{\alpha^2}{2n} \cdot \dfrac{1}{l(l+1/2)}, & j = l - 1/2 \quad (l \ge 1) \end{cases}}$$

**$l=0$ 的情况**：对于 $s$ 态，$l=0$，$\langle 1/r^3 \rangle$ 积分发散（$1/r^3$ 在原点附近的行为 $\sim 1/r^3$，但 $R_{n0} \sim$ 常数在原点，$r^2 dr$ 测度给出 $\int dr/r$ 发散）。然而，$l=0$ 时 $\langle \hat{\mathbf{L}} \cdot \hat{\mathbf{S}} \rangle = 0$（因为 $l=0$ 时 $\hat{\mathbf{L}} = 0$）。这个 $0 \cdot \infty$ 的不定式在狄拉克方程的严格解中给出有限的结果——事实上，达尔文项恰好在 $l=0$ 时发挥作用，并与自旋-轨道耦合的极限情况合并，给出正确的精细结构能级。

### 四、达尔文项

### 4.1 物理起源

相对论量子力学中，电子的位置不是一个精确的点，而有一个固有的“颤动”（Zitterbewegung），其尺度约为康普顿波长 $\hbar/(m_e c)$。这种颤动使得电子感受到的势能不是 $V(r)$，而是一个“涂抹”后的势能 $V(r + \delta \mathbf{r})$。对 $\delta \mathbf{r}$ 做平均，给出最低阶修正——**达尔文项**。

其哈密顿量为：

$$\hat{H}_{\text{Darwin}} = \frac{\hbar^2}{8 m_e^2 c^2} \nabla^2 V$$

对于库仑势 $V = -k/r$，在 $r \neq 0$ 处 $\nabla^2 V = 0$，但在原点处有奇异性。事实上，利用 $\nabla^2 (1/r) = -4\pi \delta^{(3)}(\mathbf{r})$，我们得到：

$$\hat{H}_{\text{Darwin}} = \frac{\hbar^2}{8 m_e^2 c^2} \cdot 4\pi k \delta^{(3)}(\mathbf{r}) = \frac{\pi \hbar^2 k}{2 m_e^2 c^2} \delta^{(3)}(\mathbf{r})$$

### 4.2 能量修正

达尔文项的期望值正比于电子在原点处的概率密度：

$$\Delta E_{\text{Darwin}}^{(1)} = \frac{\pi \hbar^2 k}{2 m_e^2 c^2} |\psi_{nlm}(0)|^2$$

只有 $l=0$（$s$ 态）在原点处波函数非零。对于 $s$ 态：

$$|\psi_{n00}(0)|^2 = \frac{1}{\pi n^3 a_0^3}$$

这是因为 $R_{n0}(0) = 2 (n a_0)^{-3/2}$（由径向波函数直接计算），而 $Y_{00} = 1/\sqrt{4\pi}$，所以 $|\psi(0)|^2 = |R_{n0}(0)|^2 |Y_{00}|^2 = (4/(n^3 a_0^3)) \cdot (1/(4\pi)) = 1/(\pi n^3 a_0^3)$。

代入：

$$\Delta E_{\text{Darwin}}^{(1)}(l=0) = \frac{\pi \hbar^2 k}{2 m_e^2 c^2} \cdot \frac{1}{\pi n^3 a_0^3}$$

利用 $a_0 = \hbar^2/(m_e k)$ 和 $\alpha = k/(\hbar c)$，得到：

$$\Delta E_{\text{Darwin}}^{(1)}(l=0) = \frac{\hbar^2 k}{2 m_e^2 c^2 n^3 a_0^3} = \frac{k^4 m_e}{2 \hbar^4 c^2 n^3} = -E_n \frac{\alpha^2}{n}$$

准确来说：

$$\boxed{\Delta E_{\text{Darwin}}^{(1)}(l=0) = -E_n \frac{\alpha^2}{n}}$$

对于 $l \neq 0$，$\Delta E_{\text{Darwin}}^{(1)} = 0$。

### 五、精细结构：三项修正的统一

### 5.1 三种修正的合并

精细结构由三项贡献组成：

-   相对论动能修正 $\Delta E_{\text{rel}}$
-   自旋-轨道耦合 $\Delta E_{\text{SO}}$
-   达尔文项 $\Delta E_{\text{Darwin}}$（仅 $l=0$ 非零）

将这三项相加，不同 $l$ 和 $j$ 的结果有一个简洁的统一表达式：

$$\boxed{\Delta E_{\text{FS}} = -E_n \frac{\alpha^2}{n^2} \left( \frac{n}{j+1/2} - \frac{3}{4} \right)}$$

这个公式对 $l=0$（此时 $j=1/2$）和 $l \ge 1$（$j = l \pm 1/2$）都成立。它是狄拉克方程精确解的 $O(\alpha^2)$ 展开，可以从狄拉克能级公式：

$$E_{nj} = m_e c^2 \left( \sqrt{1 + \left( \frac{\alpha}{n - (j+1/2) + \sqrt{(j+1/2)^2 - \alpha^2}} \right)^2} - 1 \right)$$

按 $\alpha^2$ 展开得到。

精细结构修正后的能级只依赖于 $n$ 和 $j$，对 $l$ 的依赖消失——具有相同 $n$ 和 $j$ 但不同 $l$ 的态（如 $2s_{1/2}$ 和 $2p_{1/2}$）在狄拉克理论中能量相同。

### 5.2 具体例子：$n=2$ 精细结构

对于 $n=2$：

-   $l=0, j=1/2$（$2s_{1/2}$）
-   $l=1, j=1/2$（$2p_{1/2}$）
-   $l=1, j=3/2$（$2p_{3/2}$）

应用精细结构公式，$E_2 = -3.4$ eV，$\alpha^2 \approx 5.33 \times 10^{-5}$：

-   $\Delta E_{\text{FS}}(2s_{1/2}) = \Delta E_{\text{FS}}(2p_{1/2}) = -E_2 \frac{\alpha^2}{4} \left( \frac{2}{1} - \frac{3}{4} \right) = -E_2 \frac{\alpha^2}{4} \cdot \frac{5}{4} = -\frac{5}{16} E_2 \alpha^2 \approx -5.66 \times 10^{-5} \,\text{eV}$
-   $\Delta E_{\text{FS}}(2p_{3/2}) = -E_2 \frac{\alpha^2}{4} \left( \frac{2}{2} - \frac{3}{4} \right) = -E_2 \frac{\alpha^2}{4} \cdot \frac{1}{4} = -\frac{1}{16} E_2 \alpha^2 \approx -1.13 \times 10^{-5} \,\text{eV}$

$2p_{3/2}$ 比 $2s_{1/2}$ 和 $2p_{1/2}$ 高出约 $4.53 \times 10^{-5}$ eV。这个能级差对应频率约为 $11$ GHz，波长约 $2.7$ cm——在微波波段。

### 六、兰姆移位：量子电动力学的印记

### 6.1 实验发现与理论来源

1947 年，兰姆和 Retherford 用微波共振技术精确测量了氢原子 $n=2$ 能级，发现 $2s_{1/2}$ 和 $2p_{1/2}$ 并不完全简并——$2s_{1/2}$ 比 $2p_{1/2}$ 高出约 $1057$ MHz（$\approx 4.37 \times 10^{-6}$ eV）。

这在狄拉克理论中是无法解释的——狄拉克方程预言 $2s_{1/2}$ 和 $2p_{1/2}$ 能量相同。

兰姆移位的理论解释来自量子电动力学（QED），主要贡献包括：

1.  **真空极化**：原子核的电场使真空中的虚电子-正电子对极化，形成屏蔽效应。
2.  **电子自能**：电子与其自身辐射的虚光子相互作用，导致电子的有效质量和能量发生变化。

### 6.2 贝特与兰姆移位计算

汉斯·贝特在 1947 年给出了兰姆移位的第一个非协变但基本正确的计算。核心思想是：电子自能与自由电子的自能之差是有限的，只要做质量重整化。

兰姆移位的 QED 计算结果（对 $n=2$）：

$$\Delta E_{\text{Lamb}}(2s_{1/2}) = \frac{\alpha^3 R_y}{3\pi} \ln\left( \frac{m_e c^2}{k_{\text{min}}} \right)$$

其中 $k_{\text{min}}$ 是一个红外截断，约为 $\sim 17.8\,R_y$（由贝特用平均激发能近似估算）。完整的 QED 计算给出：

$$\Delta E_{\text{Lamb}}(2s_{1/2}) = \frac{\alpha^3 R_y}{3\pi} \left( \ln\frac{1}{\alpha^2} + \frac{19}{30} \right) \approx 1057 \,\text{MHz}$$

$2p_{1/2}$ 态的兰姆移位远小于 $2s_{1/2}$，这是因为只有 $s$ 态电子在原点处有显著的概率密度，而虚光子效应在核附近最强。

一般地，兰姆移位主要影响 $l=0$ 态，对 $l \neq 0$ 态的影响随着 $l$ 增大而迅速减小。其对主量子数 $n$ 的依赖大致为 $\Delta E_{\text{Lamb}} \propto 1/n^3$。

### 6.3 现代精确值

经过数十年的理论和实验改进，氢原子兰姆移位的理论和实验值已经达到了极高的精度。目前，$1s$ 兰姆移位的理论值约为 $8172.8$ MHz（$\approx 3.38 \times 10^{-5}$ eV），$2s-2p$ 兰姆移位约为 $1057.845$ MHz。兰姆移位的精确测量和计算是检验 QED 的最重要手段之一。

### 七、超精细结构：电子与核自旋的耦合

### 7.1 物理机制

氢原子的原子核（质子）具有自旋 $I = 1/2$，相应的核磁矩为：

$$\hat{\boldsymbol{\mu}}_N = g_N \frac{e}{2 m_p} \hat{\mathbf{I}}$$

其中 $g_N \approx 5.586$ 是质子的 $g$ 因子（远大于 $2$，因为质子是复合粒子）。

电子与核磁矩之间的磁偶极-偶极相互作用导致**超精细结构**。哈密顿量为：

$$\hat{H}_{\text{hf}} = \frac{\mu_0}{4\pi} \left[ \frac{8\pi}{3} \hat{\boldsymbol{\mu}}_e \cdot \hat{\boldsymbol{\mu}}_N \delta^{(3)}(\mathbf{r}) + \frac{1}{r^3} \left( \hat{\boldsymbol{\mu}}_e \cdot \hat{\boldsymbol{\mu}}_N - 3 (\hat{\boldsymbol{\mu}}_e \cdot \hat{\mathbf{r}})(\hat{\boldsymbol{\mu}}_N \cdot \hat{\mathbf{r}}) \right) \right]$$

其中 $\hat{\boldsymbol{\mu}}_e = -g_s \frac{\mu_B}{\hbar} \hat{\mathbf{S}}$，$g_s \approx 2$。

对于 $s$ 态，只有 $\delta$ 函数项（费米接触项）有贡献，因为第二项在球对称的 $s$ 态下平均为零。费米接触项为：

$$\hat{H}_{\text{hf}}^{\text{Fermi}} = \frac{2}{3} \mu_0 g_s \mu_B g_N \mu_N |\psi(0)|^2 \hat{\mathbf{I}} \cdot \hat{\mathbf{S}}$$

其中 $\mu_N = e\hbar/(2m_p)$ 是核磁子。

### 7.2 总角动量与超精细能级

定义总角动量（包括核自旋）：

$$\hat{\mathbf{F}} = \hat{\mathbf{J}} + \hat{\mathbf{I}}$$

对于基态氢原子，$l=0$，$j=1/2$，$\hat{\mathbf{J}} = \hat{\mathbf{S}}$（因为 $l=0$）。此时：

$$\hat{H}_{\text{hf}} = A_{\text{hf}} \hat{\mathbf{I}} \cdot \hat{\mathbf{S}}$$

其中 $A_{\text{hf}}$ 是超精细常数。利用 $\hat{\mathbf{F}}^2 = \hat{\mathbf{I}}^2 + \hat{\mathbf{S}}^2 + 2 \hat{\mathbf{I}} \cdot \hat{\mathbf{S}}$：

$$\hat{\mathbf{I}} \cdot \hat{\mathbf{S}} = \frac{1}{2} (\hat{\mathbf{F}}^2 - \hat{\mathbf{I}}^2 - \hat{\mathbf{S}}^2)$$

在 $|F, m_F\rangle$ 基底下，本征值为：

$$\langle \hat{\mathbf{I}} \cdot \hat{\mathbf{S}} \rangle = \frac{\hbar^2}{2} \left[ F(F+1) - I(I+1) - s(s+1) \right]$$

$I=1/2$，$s=1/2$，因此 $F = 0, 1$。

-   $F = 1$：$\langle \hat{\mathbf{I}} \cdot \hat{\mathbf{S}} \rangle = \frac{\hbar^2}{2}(2 - 3/4 - 3/4) = \frac{\hbar^2}{4}$
-   $F = 0$：$\langle \hat{\mathbf{I}} \cdot \hat{\mathbf{S}} \rangle = \frac{\hbar^2}{2}(0 - 3/4 - 3/4) = -\frac{3\hbar^2}{4}$

能量修正为 $\Delta E_{\text{hf}} = A_{\text{hf}} \langle \hat{\mathbf{I}} \cdot \hat{\mathbf{S}} \rangle / \hbar^2$。

### 7.3 基态超精细分裂与 21 cm 线

对于基态氢原子（$n=1, l=0$），超精细常数为：

$$A_{\text{hf}} = \frac{8}{3} \pi \mu_0 g_s \mu_B g_N \mu_N |\psi_{100}(0)|^2$$

基态波函数在原点的值：

$$|\psi_{100}(0)|^2 = \frac{1}{\pi a_0^3}$$

代入所有常数，得到基态超精细分裂：

$$\Delta E_{\text{hf}}(F=1) - \Delta E_{\text{hf}}(F=0) = A_{\text{hf}} \left( \frac{1}{4} - \left(-\frac{3}{4}\right) \right) = A_{\text{hf}} \hbar^2$$

数值上，$A_{\text{hf}} \approx 5.9 \times 10^{-6}$ eV，对应的频率为：

$$\boxed{\nu_{21\text{cm}} = 1420.4057517667 \,\text{MHz}}$$

波长约 $21.1$ cm。这就是著名的**21 cm 线**，它来自氢原子基态的超精细能级跃迁（$F=1 \to F=0$ 加上一个光子的自发辐射）。这条谱线在射电天文学中具有极其重要的地位——它是探测星际中性氢的主要手段，绘制了银河系旋臂结构，并为宇宙大尺度结构的研究提供了关键数据。

### 7.4 激发态的超精细结构

对于 $l \neq 0$ 的态，超精细相互作用包含费米接触项和磁偶极-偶极项，计算更为复杂。一般地，超精细常数 $A_{\text{hf}}$ 随 $n$ 和 $l$ 的增大而迅速减小。例如，$2p_{3/2}$ 态的超精细分裂远小于基态，但它对高精度光谱学和量子电动力学的检验同样重要。

* * *

### 八、外场效应：塞曼效应与斯塔克效应

### 8.1 塞曼效应

在外部均匀磁场 $\mathbf{B} = B \hat{z}$ 中，氢原子的能级进一步分裂。哈密顿量为：

$$\hat{H}_Z = \frac{\mu_B B}{\hbar} (\hat{L}_z + g_s \hat{S}_z)$$

弱场下（精细结构未被破坏），在 $|n, l, j, m_j\rangle$ 基底下，塞曼效应一阶修正由投影定理给出：

$$\Delta E_Z = \mu_B B g_J m_j$$

其中朗德 $g$ 因子为：

$$g_J = 1 + \frac{j(j+1) - l(l+1) + s(s+1)}{2j(j+1)}$$

对于 $j = l \pm 1/2$：

$$g_J = \begin{cases} \frac{2l+2}{2l+1}, & j = l + 1/2 \\[1em] \frac{2l}{2l+1}, & j = l - 1/2 \end{cases}$$

强场下（帕邢-巴克效应），$\hat{\mathbf{L}}$ 和 $\hat{\mathbf{S}}$ 与磁场耦合，基底变为 $|n, l, m_l, m_s\rangle$，能级分裂由 $\mu_B B (m_l + g_s m_s)$ 给出。

### 8.2 斯塔克效应

在外部均匀电场 $\mathbf{E} = E \hat{z}$ 中，氢原子能级分裂（斯塔克效应）。哈密顿量为：

$$\hat{H}_S = e E z$$

由于氢原子具有 $l$ 简并（不同 $l$ 的态能量相同），一阶斯塔克效应在简并微扰论框架下处理，能级分裂线性依赖于 $E$。例如，$n=2$ 的四个态中，$2s$ 和 $2p_0$ 在电场中混合，产生线性的能级分裂。对于非简并态（如基态），一阶斯塔克效应为零（因为宇称对称性），只有二阶斯塔克效应（能级移动 $\propto E^2$）。

* * *

### 九、其他微小修正

现代氢原子光谱学还涉及多项更精细的修正：

1.  **核反冲效应**：氢原子核（质子）的质量有限，导致折合质量修正 $R_M = R_\infty / (1 + m_e/m_p)$。这使得氘和氢的里德伯常数有微小差异（同位素移位）。
2.  **核有限体积效应**：原子核不是点电荷，其电荷分布对 $s$ 态（特别是重核）的能级有微小影响。对于氢，质子电荷半径约 $0.84$ fm，导致 $1s$ 兰姆移位修正约 $1.4$ MHz。
3.  **双光子交换与更高阶 QED**：除了单圈（$\alpha^3$）的兰姆移位，双圈（$\alpha^4$）和更高阶的 QED 修正（约几十 kHz 级别）在现代氢原子光谱的精确理论中必须考虑。
4.  **弱相互作用**：电子与核之间的宇称不守恒弱相互作用（$Z$ 玻色子交换）在氢原子中产生极微小的能级移动（远小于目前实验精度），但它在重原子中可被探测。

* * *

### 十、修正总结与能级全景图

将上述所有修正按能量尺度从大到小排列：

| 修正类型 | 量级（eV） | 物理来源 |
| ----- | ----- | ----- |
| 非相对论薛定谔能级 | 1 \sim 10 | 库仑势 |
| 精细结构 | \sim 10^{-4} | 相对论 p^4 + 自旋-轨道 + 达尔文 |
| 超精细结构 | \sim 10^{-6} | 电子-核自旋磁相互作用 |
| 兰姆移位 | \sim 10^{-6}（s 态） | QED 辐射修正 |
| 塞曼/斯塔克效应 | 可变（外场） | 外磁场/外电场 |
| 核有限体积、反冲等 | \sim 10^{-9} \sim 10^{-7} | 核结构与运动学 |

氢原子完整的基态能级可以表示为：

$$E_{1s} = -\frac{R_y}{1 + m_e/m_p} \left[ 1 - \frac{\alpha^2}{4} + \cdots \right] + \Delta E_{\text{Lamb}} + \Delta E_{\text{hf}}$$

每一项都代表着一个物理学分支的基石：非相对论量子力学、相对论量子力学、量子电动力学，以及核物理。

* * *

### 小结

从基础的薛定谔解出发，我们系统地引入了氢原子模型的多层修正：

1.  **相对论动能修正**（$p^4$ 项）和**自旋-轨道耦合**带来了精细结构，使得能级依赖于总角动量 $j$。狄拉克方程将这三项与**达尔文项**统一为简洁的 $\alpha^2$ 展开公式。
2.  **兰姆移位**（$2s_{1/2} - 2p_{1/2}$ 约 $1057$ MHz）揭示了电子与真空涨落的相互作用，是量子电动力学的实验基础。
3.  **超精细结构**来自电子与核磁矩的耦合，基态 $F=0 \leftrightarrow 1$ 分裂对应著名的 $21$ cm 线，是射电天文学的核心工具。
4.  **外场效应**（塞曼和斯塔克）为原子光谱学提供了丰富的诊断手段。
5.  现代氢原子光谱学已进入 $10^{-15}$ 的相对精度，氢原子成为检验基本物理定律、测量基本常数、甚至探测新物理的精密实验室。

氢原子从最简单的薛定谔方程出发，一层层加上修正，每一步都对应着物理学中一个深刻的新原理。它不只是一个原子——它是相对论、量子场论和原子核物理的交叉路口，是精密检验自然律的终极竞技场。
