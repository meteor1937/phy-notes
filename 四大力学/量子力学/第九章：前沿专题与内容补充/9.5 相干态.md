在谐振子的量子理论中，我们遇到了两套基底：一套是能量本征态——数态 $|n\rangle$，它们有确定的能量，但位置和动量的期望值都为零，是一些“静止的”量子态；另一套是我们在波包理论中构造的高斯波包，它们有大致确定的位置和动量，会像经典粒子一样运动，但在自由场中会不可避免地扩散。

一个自然的问题是：**在谐振子势中，是否存在一种态，它既像经典粒子那样做周期性振荡，又不会像自由波包那样随时间扩散？**

答案是肯定的。这就是**相干态**。相干态是最接近经典谐振子的量子态——它的波包形状永不改变，它的期望值精确地遵循经典运动方程，它的量子不确定性始终保持在基态的极小值。相干态是量子光学中激光理论的数学基础，是介观超导电路中“量子比特”的操控语言，更是连接量子世界与经典世界的概念桥梁。

* * *

### 一、相干态的定义：湮灭算符的本征态

数态 $|n\rangle$ 是数算符 $\hat{N} = \hat{a}^\dagger \hat{a}$ 的本征态。那湮灭算符 $\hat{a}$ 本身有没有本征态？由于 $\hat{a}$ 不是厄米算符，它的本征值可以是复数。

**定义**：相干态 $|\alpha\rangle$ 是湮灭算符 $\hat{a}$ 的本征态，本征值为复数 $\alpha$：

$$\boxed{\hat{a} |\alpha\rangle = \alpha |\alpha\rangle}$$

其中 $\alpha \in \mathbb{C}$ 是一个任意复数，称为**相干参数**。由于 $\hat{a}$ 非厄米，相干态不是正交的（后面会细讲），但它们构成一个超完备的基底。

这个看似简单的定义蕴含着极其丰富的物理内容。让我们一步一步将它展开。

* * *

### 二、相干态在数态基底下的展开

数态 $\{|n\rangle\}$ 构成态空间的一组正交归一基。我们可以把相干态在这组基下展开：

$$|\alpha\rangle = \sum_{n=0}^{\infty} c_n |n\rangle$$

代入本征方程 $\hat{a} |\alpha\rangle = \alpha |\alpha\rangle$：

$$\hat{a} \sum_{n=0}^{\infty} c_n |n\rangle = \sum_{n=1}^{\infty} c_n \sqrt{n} |n-1\rangle = \alpha \sum_{n=0}^{\infty} c_n |n\rangle$$

将左边求和指标换为 $m = n-1$：

$$\sum_{m=0}^{\infty} c_{m+1} \sqrt{m+1} |m\rangle = \alpha \sum_{n=0}^{\infty} c_n |n\rangle$$

比较 $|n\rangle$ 的系数，得到递推关系：

$$c_{n+1} \sqrt{n+1} = \alpha c_n \quad \Rightarrow \quad c_{n+1} = \frac{\alpha}{\sqrt{n+1}} c_n$$

迭代得到：

$$c_n = \frac{\alpha^n}{\sqrt{n!}} c_0$$

因此：

$$|\alpha\rangle = c_0 \sum_{n=0}^{\infty} \frac{\alpha^n}{\sqrt{n!}} |n\rangle$$

归一化条件 $\langle \alpha | \alpha \rangle = 1$ 确定 $c_0$：

$$\langle \alpha | \alpha \rangle = |c_0|^2 \sum_{n=0}^{\infty} \frac{|\alpha|^{2n}}{n!} = |c_0|^2 e^{|\alpha|^2} = 1$$

取 $c_0 = e^{-|\alpha|^2/2}$（正实数相位约定）。于是相干态在数态基下的归一化展开为：

$$\boxed{|\alpha\rangle = e^{-|\alpha|^2/2} \sum_{n=0}^{\infty} \frac{\alpha^n}{\sqrt{n!}} |n\rangle}$$

这是一个优美的结果。相干态是**所有数态的无限叠加**，叠加系数由泊松分布给出。

* * *

### 三、相干态的粒子数统计性质

从展开式可以直接读出相干态中的**粒子数分布**。测量数算符 $\hat{N}$ 得到结果 $n$ 的概率为：

$$P(n) = |\langle n | \alpha \rangle|^2 = e^{-|\alpha|^2} \frac{|\alpha|^{2n}}{n!}$$

这正是**泊松分布**，其均值和方差为：

$$\langle \hat{N} \rangle = |\alpha|^2, \quad \Delta N = \sqrt{\langle \hat{N}^2 \rangle - \langle \hat{N} \rangle^2} = |\alpha|$$

所以 $|\alpha|^2$ 是平均粒子数，相对涨落 $\Delta N / \langle N \rangle = 1/|\alpha|$。当 $|\alpha| \gg 1$ 时，相对涨落很小——相干态具有近似确定的“粒子数”，但又不是精确的本征态。

这个泊松统计是相干态的标志性特征。它直接对应激光的光子统计——远高于阈值时，激光的光子数分布就是泊松分布。热光场的粒子数则服从玻色-爱因斯坦分布（超泊松统计）。

* * *

### 四、相干态在坐标表象中的波函数

我们来求相干态在坐标表象下的波函数 $\psi_\alpha(x) = \langle x | \alpha \rangle$。

基态 $|0\rangle$ 的波函数已知为高斯函数。利用相干态可以写为平移的基态（见下一节），但更直接的方法是从定义出发。

湮灭算符在坐标表象中为：

$$\hat{a} = \frac{1}{\sqrt{2}} \left( \sqrt{\frac{m\omega}{\hbar}} x + \frac{\hbar}{\sqrt{m\hbar\omega}} \frac{d}{dx} \right) = \frac{1}{\sqrt{2}} \left( \xi + \frac{d}{d\xi} \right)$$

其中 $\xi = \sqrt{m\omega/\hbar} \, x$ 是无量纲坐标。

本征方程 $\hat{a} \psi_\alpha = \alpha \psi_\alpha$ 写成：

$$\frac{1}{\sqrt{2}} \left( \xi + \frac{d}{d\xi} \right) \psi_\alpha(\xi) = \alpha \psi_\alpha(\xi)$$

即：

$$\frac{d\psi_\alpha}{d\xi} = (\sqrt{2}\alpha - \xi) \psi_\alpha$$

这个一阶微分方程的解为：

$$\psi_\alpha(\xi) = A \exp\left( -\frac{\xi^2}{2} + \sqrt{2}\alpha \xi \right)$$

将 $\alpha$ 分解为实部和虚部：$\alpha = \alpha_R + i\alpha_I$。回到有量纲坐标 $x$：

$$\psi_\alpha(x) = A \exp\left[ -\frac{m\omega}{2\hbar} x^2 + \sqrt{\frac{2m\omega}{\hbar}} \alpha x \right]$$

配方指数上的二次型，令 $\alpha = |\alpha| e^{i\theta}$，并定义经典振幅 $x_0 = \sqrt{2\hbar/(m\omega)} \, \text{Re}(\alpha) = \sqrt{2\hbar/(m\omega)} \, \alpha_R$，$p_0 = \sqrt{2m\hbar\omega} \, \text{Im}(\alpha) = \sqrt{2m\hbar\omega} \, \alpha_I$。

归一化后得到：

$$\boxed{\psi_\alpha(x) = \left( \frac{m\omega}{\pi\hbar} \right)^{1/4} \exp\left[ -\frac{m\omega}{2\hbar} \left( x - \sqrt{\frac{2\hbar}{m\omega}} \alpha_R \right)^2 + i \sqrt{\frac{2m\omega}{\hbar}} \alpha_I x - i \alpha_R \alpha_I \right]}$$

这是一般的含时相干态波函数（静止形式）。下一节我们会看到，这个波函数是一个位移了的高斯函数——与基态形状完全相同，只是中心移到了 $x_0$，并赋予了一个动量 $p_0$。

* * *

### 五、相干态的位移算符表示

相干态有一个极其重要的等价定义：**相干态是基态经过相空间位移后的结果**。

定义**位移算符**：

$$\boxed{\hat{D}(\alpha) \equiv e^{\alpha \hat{a}^\dagger - \alpha^* \hat{a}}}$$

这个算符的作用是将相空间中的态整体平移。利用贝克-坎贝尔-豪斯多夫公式（因为 $[\hat{a}, \hat{a}^\dagger] = 1$ 是常数，嵌套对易子到此为止），$\hat{D}(\alpha)$ 可以分解为：

$$\hat{D}(\alpha) = e^{-|\alpha|^2/2} e^{\alpha \hat{a}^\dagger} e^{-\alpha^* \hat{a}}$$

现在用 $\hat{D}(\alpha)$ 作用在基态 $|0\rangle$ 上：

$$\hat{D}(\alpha) |0\rangle = e^{-|\alpha|^2/2} e^{\alpha \hat{a}^\dagger} \underbrace{e^{-\alpha^* \hat{a}} |0\rangle}_{=|0\rangle} = e^{-|\alpha|^2/2} e^{\alpha \hat{a}^\dagger} |0\rangle$$

展开 $e^{\alpha \hat{a}^\dagger}$：

$$e^{\alpha \hat{a}^\dagger} |0\rangle = \sum_{n=0}^{\infty} \frac{\alpha^n}{n!} (\hat{a}^\dagger)^n |0\rangle = \sum_{n=0}^{\infty} \frac{\alpha^n}{\sqrt{n!}} |n\rangle$$

因此：

$$\hat{D}(\alpha) |0\rangle = e^{-|\alpha|^2/2} \sum_{n=0}^{\infty} \frac{\alpha^n}{\sqrt{n!}} |n\rangle = |\alpha\rangle$$

**相干态就是平移后的基态**：

$$\boxed{|\alpha\rangle = \hat{D}(\alpha) |0\rangle}$$

这个定义的物理图像极为清晰：基态 $|0\rangle$ 是相空间中以原点为中心的高斯波包。位移算符 $\hat{D}(\alpha)$ 将整个波包平移到相空间中的点 $(\text{Re}(\alpha), \text{Im}(\alpha))$，而不改变其形状。

将 $\alpha$ 参数化为相空间坐标：

$$\alpha = \sqrt{\frac{m\omega}{2\hbar}} x_0 + i \frac{p_0}{\sqrt{2m\hbar\omega}}$$

其中 $x_0 = \langle \hat{x} \rangle$ 是位置期望值，$p_0 = \langle \hat{p} \rangle$ 是动量期望值。相干态由这两个经典相空间坐标完全确定。

* * *

### 六、相干态的最小不确定度性质

相干态的量子涨落保持在基态的最小水平。

在相干态下计算位置和动量的方差。利用 $\hat{x} = \sqrt{\hbar/(2m\omega)} (\hat{a} + \hat{a}^\dagger)$，$\hat{p} = -i \sqrt{m\hbar\omega/2} (\hat{a} - \hat{a}^\dagger)$：

$$\langle \alpha | \hat{x} | \alpha \rangle = \sqrt{\frac{\hbar}{2m\omega}} (\alpha + \alpha^*) = \sqrt{\frac{2\hbar}{m\omega}} \, \text{Re}(\alpha) = x_0$$

$$\langle \alpha | \hat{x}^2 | \alpha \rangle = \frac{\hbar}{2m\omega} \langle (\hat{a} + \hat{a}^\dagger)^2 \rangle = \frac{\hbar}{2m\omega} ( (\alpha + \alpha^*)^2 + 1 )$$

因此：

$$\Delta x = \sqrt{\langle \hat{x}^2 \rangle - \langle \hat{x} \rangle^2} = \sqrt{\frac{\hbar}{2m\omega}}$$

类似地：

$$\Delta p = \sqrt{\frac{m\hbar\omega}{2}}$$

所以：

$$\boxed{\Delta x \cdot \Delta p = \frac{\hbar}{2}}$$

**相干态满足最小不确定度乘积**，与基态相同。这不是偶然的——位移算符 $\hat{D}(\alpha)$ 是幺正的，它只是把整个概率分布平移到相空间的另一个位置，不改变分布的宽度。相干态就是“移到了别处”的基态。

**相干态的相空间图像**：在相空间中，基态是一个以原点为中心的圆形高斯分布（$\Delta x = \Delta p / (m\omega)$ 在适当标度下相等）。相干态是将这个圆形高斯分布平移到 $(\alpha_R, \alpha_I)$ 处，形状、大小完全不变。量子涨落以这个最小不确定度的“模糊圆盘”来体现——这是经典点粒子在量子力学中所能接近的极限。

* * *

### 七、相干态的时间演化：永不扩散的波包

这是相干态最引人注目的性质。我们来计算相干态在谐振子哈密顿量下的时间演化。

初始相干态 $|\alpha(0)\rangle = |\alpha_0\rangle$。时间演化由 $\hat{H} = \hbar\omega(\hat{a}^\dagger \hat{a} + 1/2)$ 驱动。在海森堡绘景中，湮灭算符的时间演化是 $\hat{a}(t) = \hat{a} e^{-i\omega t}$。在薛定谔绘景中，态演化：

$$|\alpha(t)\rangle = e^{-i\hat{H}t/\hbar} |\alpha_0\rangle$$

将相干态的数态展开代入，利用 $e^{-i\hat{H}t/\hbar} |n\rangle = e^{-i\omega(n+1/2)t} |n\rangle$：

$$|\alpha(t)\rangle = e^{-|\alpha_0|^2/2} \sum_{n=0}^{\infty} \frac{\alpha_0^n}{\sqrt{n!}} e^{-i\omega(n+1/2)t} |n\rangle$$

整理指数：

$$|\alpha(t)\rangle = e^{-i\omega t/2} e^{-|\alpha_0|^2/2} \sum_{n=0}^{\infty} \frac{(\alpha_0 e^{-i\omega t})^n}{\sqrt{n!}} |n\rangle$$

这正是相干态 $|\alpha(t)\rangle = e^{-i\omega t/2} |\alpha_0 e^{-i\omega t}\rangle$（整体相位 $e^{-i\omega t/2}$ 可忽略）。因此：

$$\boxed{\alpha(t) = \alpha_0 e^{-i\omega t}}$$

**相干态在时间演化下保持为相干态**，其相干参数在复平面上以角频率 $\omega$ 顺时针旋转。这个简单的演化规律意味着：

**位置和动量的期望值精确遵循经典运动方程**：

$$\langle \hat{x} \rangle(t) = x_0 \cos(\omega t) + \frac{p_0}{m\omega} \sin(\omega t)$$

$$\langle \hat{p} \rangle(t) = -m\omega x_0 \sin(\omega t) + p_0 \cos(\omega t)$$

**波包永不扩散**：波包的概率密度为：

$$|\psi_\alpha(x,t)|^2 = \sqrt{\frac{m\omega}{\pi\hbar}} \exp\left[ -\frac{m\omega}{\hbar} \left( x - \langle \hat{x} \rangle(t) \right)^2 \right]$$

这是一个以 $\langle \hat{x} \rangle(t)$ 为中心、恒定宽度 $\sqrt{\hbar/(2m\omega)}$ 的高斯波包。它像经典粒子一样来回振荡，但形状永远不变。

为什么谐振子中的相干态不扩散，而自由高斯波包会扩散？根源在于色散关系。自由粒子的 $\omega(k) \propto k^2$ 导致不同 $k$ 分量的相速度不同，波包必然扩散。谐振子势将所有本征模式的频率锁定为 $\omega_n = \omega(n+1/2)$——等间距能级意味着所有傅里叶分量都以相同的频率 $\omega$ 的整数倍振荡，相位演化是同步的（除整体周期外）。恢复力精确地平衡了色散，使波包保持其形状。

### 八、相干态的数学性质：超完备性与非正交性

### 8.1 非正交性

两个不同相干态的内积不为零：

$$\langle \beta | \alpha \rangle = e^{-|\alpha|^2/2 - |\beta|^2/2} \sum_{n,m} \frac{(\beta^*)^m \alpha^n}{\sqrt{m! n!}} \langle m | n \rangle = e^{-|\alpha|^2/2 - |\beta|^2/2} \sum_{n=0}^{\infty} \frac{(\beta^* \alpha)^n}{n!}$$

即：

$$\boxed{\langle \beta | \alpha \rangle = e^{-\frac{1}{2}(|\alpha|^2 + |\beta|^2) + \beta^* \alpha}}$$

$$|\langle \beta | \alpha \rangle|^2 = e^{-|\alpha - \beta|^2}$$

当 $\alpha$ 和 $\beta$ 差别很大时（$|\alpha - \beta| \gg 1$），内积指数衰减——宏观上不同的相干态近似正交。但严格来说，它们总是有非零的内积。

### 8.2 超完备性

尽管相干态不是正交的，它们却构成一个**超完备基**。这意味着任何态都可以用相干态展开，但展开方式不唯一。关键恒等式是**单位算符的分解**：

$$\boxed{\frac{1}{\pi} \int d^2\alpha \, |\alpha\rangle \langle \alpha| = \hat{I}}$$

其中 $d^2\alpha = d(\text{Re}\,\alpha) \, d(\text{Im}\,\alpha)$。积分遍及整个复 $\alpha$ 平面。

**证明**：用极坐标 $\alpha = r e^{i\theta}$，数态展开代入：

$$\frac{1}{\pi} \int d^2\alpha \, |\alpha\rangle \langle \alpha| = \frac{1}{\pi} \int_0^{\infty} r dr \int_0^{2\pi} d\theta \, e^{-r^2} \sum_{n,m} \frac{r^{n+m} e^{i(n-m)\theta}}{\sqrt{n! m!}} |n\rangle \langle m|$$

$\theta$ 积分给出 $2\pi \delta_{n,m}$，因此：

$$= 2 \int_0^{\infty} dr \, r e^{-r^2} \sum_{n} \frac{r^{2n}}{n!} |n\rangle \langle n| = \sum_n |n\rangle \langle n| = \hat{I}$$

这个单位算符分解是相干态路径积分和量子光学中相空间方法（如胡斯米 $Q$ 函数、维格纳函数）的数学基础。

* * *

### 九、相干态在量子光学中的应用纲要

相干态的物理意义在量子光学中得到了最充分的体现。

**激光的量子描述**：理想单模激光（远高于阈值）的量子态就是相干态。光子数服从泊松分布，相位近似确定。这与热光的超泊松统计和单光子源的反聚束统计形成鲜明对比。

**位移算符的物理实现**：一个经典的振荡电流源会产生相干态电磁场。数学上，经典的源项 $J(t)$ 与电磁场耦合，其演化算符正是 $\hat{D}(\alpha)$。

**相干态的衰减与量子噪声**：当相干态通过分束器或经历损耗时，它会保持为相干态，只是振幅衰减（$\alpha \to \sqrt{\eta}\alpha$）。这一性质使相干态成为量子通信和量子密码学中理想的“经典参考”。

* * *

### 十、相干态在路径积分和量子-经典对应中的应用

相干态的超完备性使其成为构造路径积分的理想基底。在**相干态路径积分**中，传播子写为：

$$\langle \alpha_f | e^{-i\hat{H}T/\hbar} | \alpha_i \rangle = \int \mathcal{D}[\alpha(t)] \, e^{\frac{i}{\hbar} S[\alpha(t)]}$$

其中作用量 $S[\alpha]$ 是经典谐振子作用量在复坐标下的形式。这是量子场论中功能积分方法的基础。

**经典极限**：当 $|\alpha| \gg 1$ 时，泊松分布的相对宽度 $1/|\alpha|$ 很小，位置和动量的量子涨落相对于期望值可以忽略。相干态的行为趋于经典谐振子。$\hbar \to 0$ 极限下，相干态的模糊圆盘收缩为一个点——恢复经典力学的确定性。相干态提供了**量子-经典过渡的最平滑的实现方式**，这也是它在量子测量和退相干理论中被广泛研究的原因。

* * *

### 小结

从湮灭算符的本征方程出发，我们系统地建立了相干态的完整理论：

1.  **定义与展开**：相干态 $|\alpha\rangle$ 满足 $\hat{a}|\alpha\rangle = \alpha|\alpha\rangle$。在数态基下，它呈泊松分布展开 $|\alpha\rangle = e^{-|\alpha|^2/2} \sum_n \frac{\alpha^n}{\sqrt{n!}} |n\rangle$，平均粒子数 $\langle N \rangle = |\alpha|^2$。
2.  **位移算符表示**：$|\alpha\rangle = \hat{D}(\alpha) |0\rangle$，相干态是基态在相空间中的刚性平移。这直接解释了它满足最小不确定度乘积 $\Delta x \Delta p = \hbar/2$——涨落与基态相同，与 $\alpha$ 无关。
3.  **时间演化**：在谐振子势中，$\alpha(t) = \alpha_0 e^{-i\omega t}$，波包以经典频率振荡且**永不扩散**——这是等间距能级导致的相位同步现象。位置和动量的期望值精确遵循经典运动方程。
4.  **数学结构**：相干态非正交（$|\langle \beta|\alpha\rangle|^2 = e^{-|\alpha-\beta|^2}$）但超完备（$\frac{1}{\pi}\int d^2\alpha |\alpha\rangle\langle\alpha| = \hat{I}$）。
5.  **物理应用**：相干态是理想激光的量子描述，是量子光学和量子信息的基本工具。它提供了从量子到经典的最平滑过渡（$|\alpha| \to \infty$ 或 $\hbar \to 0$）。

相干态是量子力学中最接近“经典粒子”的量子态——但不是因为它没有涨落，而是因为它的涨落是可能的最小值，且这个最小值不随时间增长。它是波粒二象性最和谐的统一：波包的形状体现了“波”的广延性，波包中心的轨道体现了“粒子”的局域性。从这个意义上说，相干态不是量子力学的特例，而是量子力学在经典极限下的**范式实现**。
