## 2.6 一维散射问题

前面两节我们处理了一维束缚态——无限深方势阱和谐振子。在这类问题中，粒子被势场约束在有限空间区域内，波函数在无穷远处指数衰减，能谱是分立的。现在我们将面对另一类重要的量子力学问题：**散射**。在散射问题中，粒子从无穷远处入射，与局域势场相互作用后，被反射或透射。能量 $E > 0$ 是预先给定的连续参数（而不是待求的本征值），波函数在无穷远处为平面波，边界条件用于确定反射和透射振幅。

散射理论是连接量子力学与实验的核心纽带——从粒子加速器中的碰撞到扫描隧道显微镜中的电子隧穿，其理论基础都建立在本节所讨论的一维散射之上。


### 一、散射问题的基本设定

#### 1.1 物理图像

考虑一个质量为 $m$ 的粒子，从左侧无穷远处入射，遇到一个局域在空间有限区域内的势场 $V(x)$。在远离势场的区域（渐近区），势场趋于常数（通常取为零）。入射粒子有一定的概率被势场反射回左侧，也有一定的概率穿透势场继续向右传播。

散射问题的标准边界条件——“出射波”条件——是：在势场的左侧，同时存在入射波（向右传播）和反射波（向左传播）；在势场的右侧，只存在透射波（向右传播），没有从右边无穷远来的向左传播的波。这个条件体现了物理上的因果律：粒子从左侧入射是“因”，反射和透射是“果”。

#### 1.2 能量条件与波数

散射态的能量 $E > 0$ 是连续取值的。对于自由粒子，能量与波数的关系为 $E = \hbar^2 k^2/(2m)$，其中 $k = \sqrt{2mE}/\hbar > 0$。在势场 $V(x)$ 的区域，局域波数取决于 $E - V(x)$ 的符号：如果 $E > V(x)$，波数是实数，波函数是振荡型的；如果 $E < V(x)$，波数变为虚数，波函数是指数衰减或增长型的。

#### 1.3 概率流密度与散射系数

在 2.2 节中，我们定义了一维概率流密度：

$$
j = \frac{\hbar}{2mi}\left( \psi^* \frac{d\psi}{dx} - \psi \frac{d\psi^*}{dx} \right) = \frac{\hbar}{m} \text{Im}\left( \psi^* \frac{d\psi}{dx} \right)
$$

对于向右传播的平面波 $\psi = A e^{i k x}$（$k > 0$），其概率流密度为：

$$
j = \frac{\hbar k}{m} |A|^2
$$

对于向左传播的平面波 $\psi = A e^{-i k x}$，其概率流密度为 $j = -(\hbar k/m)|A|^2$。

在散射问题中，入射概率流 $j_{\text{inc}}$、反射概率流 $j_{\text{ref}}$ 和透射概率流 $j_{\text{trans}}$ 分别正比于入射、反射和透射振幅的模方。**反射系数** $R$ 和**透射系数** $T$ 定义为反射和透射概率流与入射概率流之比：

$$
\boxed{R \equiv \frac{|j_{\text{ref}}|}{j_{\text{inc}}} = \frac{|B|^2}{|A|^2}}, \quad \boxed{T \equiv \frac{j_{\text{trans}}}{j_{\text{inc}}}}
$$

其中 $A$ 是入射振幅，$B$ 是反射振幅。概率守恒（连续性方程）要求入射概率流等于反射概率流与透射概率流之和，即 $R + T = 1$。这一关系是检验散射计算正确性的重要判据。


### 二、势阶散射

#### 2.1 物理模型与薛定谔方程

势阶是最简单的非平凡散射势。势能函数为：

$$
V(x) = \begin{cases}
0, & x < 0 \\
V_0, & x \ge 0
\end{cases}
$$

其中 $V_0 > 0$ 是阶高。粒子从左侧入射，能量为 $E$。我们分别处理 $E > V_0$ 和 $E < V_0$ 两种情况。

#### 2.2 $E > V_0$：经典允许区在两个区域

**左侧区域（$x < 0$）**：$V(x) = 0$，定态薛定谔方程的解为振荡型。波数 $k = \sqrt{2mE}/\hbar$。波函数包含入射波和反射波：

$$
\psi_L(x) = A e^{i k x} + B e^{-i k x}
$$

**右侧区域（$x \ge 0$）**：$V(x) = V_0$。由于 $E > V_0$，右侧也是经典允许区。波数 $q = \sqrt{2m(E - V_0)}/\hbar$（$q > 0$）。根据“出射波”条件，右侧只有向右传播的透射波（没有从 $+\infty$ 入射的向左波）：

$$
\psi_R(x) = C e^{i q x}
$$

**边界条件**：在 $x = 0$ 处，势能有有限跳跃，波函数及其一阶导数必须连续。

波函数连续：$\psi_L(0) = \psi_R(0) \Rightarrow A + B = C$。

导数连续：$\psi_L'(0) = \psi_R'(0) \Rightarrow i k(A - B) = i q C$。

联立两式。由导数条件得 $A - B = \frac{q}{k} C$。将 $C = A + B$ 代入：$A - B = \frac{q}{k}(A + B)$。整理：$A(1 - \frac{q}{k}) = B(1 + \frac{q}{k})$，因此：

$$
B = \frac{k - q}{k + q} A
$$

将 $B$ 代回 $C = A + B$：

$$
C = A + \frac{k - q}{k + q} A = \frac{2k}{k + q} A
$$

**反射系数**：入射和反射在同一区域（波数相同，$k$），因此 $R = |B|^2/|A|^2 = \left(\frac{k - q}{k + q}\right)^2$。

**透射系数**：入射和透射在不同区域，波数不同。概率流正比于波数乘以振幅模方。因此：

$$
T = \frac{j_{\text{trans}}}{j_{\text{inc}}} = \frac{q |C|^2}{k |A|^2} = \frac{q}{k} \left( \frac{2k}{k + q} \right)^2 = \frac{4kq}{(k + q)^2}
$$

可以验证 $R + T = \frac{(k-q)^2 + 4kq}{(k+q)^2} = \frac{k^2 + 2kq + q^2}{(k+q)^2} = 1$。概率守恒成立。

**物理讨论**：即使 $E > V_0$ ——经典粒子可以毫无阻碍地越过势阶——量子力学中粒子仍有非零的反射概率。这类似于光在两种折射率不同的介质界面上部分反射——量子波在“波阻抗”不匹配处发生反射。

#### 2.3 $E < V_0$：经典禁区

当 $E < V_0$ 时，右侧是经典禁区——经典粒子完全不能进入。在量子力学中，右侧的波数变为虚数：$q = \sqrt{2m(E - V_0)}/\hbar = i \kappa$，其中 $\kappa = \sqrt{2m(V_0 - E)}/\hbar > 0$。

右侧薛定谔方程的通解为 $e^{-\kappa x}$ 和 $e^{\kappa x}$。物理边界条件（$x \to \infty$ 时波函数必须有限）排除指数增长项，只保留衰减解：

$$
\psi_R(x) = C e^{-\kappa x}
$$

边界条件在 $x = 0$ 处：波函数连续 $A + B = C$；导数连续 $i k (A - B) = -\kappa C$。

由导数条件：$A - B = \frac{i \kappa}{k} C = i \frac{\kappa}{k} C$。将 $C = A + B$ 代入：

$$
A - B = i \frac{\kappa}{k}(A + B)
$$

整理：$A(1 - i\frac{\kappa}{k}) = B(1 + i\frac{\kappa}{k})$，因此：

$$
B = \frac{1 - i\kappa/k}{1 + i\kappa/k} A = \frac{k - i\kappa}{k + i\kappa} A
$$

**反射系数**：$R = \frac{|B|^2}{|A|^2} = \left|\frac{k - i\kappa}{k + i\kappa}\right|^2$。注意分子和分母互为复共轭，因此模方之比为 $1$：

$$
R = 1
$$

粒子被完全反射。

**透射系数**：右侧波函数是纯指数衰减的（非振荡型）。概率流密度 $j_R = \frac{\hbar}{2mi}(\psi_R^* \psi_R' - \psi_R \psi_R'^*)$。由于 $\psi_R = C e^{-\kappa x}$ 是实函数（$C$ 可取为实数），$\psi_R^*\psi_R' - \psi_R\psi_R'^* = C e^{-\kappa x}(-\kappa C e^{-\kappa x}) - C e^{-\kappa x}(-\kappa C e^{-\kappa x}) = 0$。因此 $j_R = 0$，$T = 0$。

但波函数在经典禁区并不为零——它以 $e^{-\kappa x}$ 的形式渗入势阶，渗入的深度约为 $1/\kappa$。这是量子力学中隧穿效应的基础。


### 三、一维方势垒与量子隧穿

#### 3.1 物理模型与薛定谔方程

现在处理一维散射中最核心的问题——方势垒。势能函数为：

$$
V(x) = \begin{cases}
0, & x < 0 \quad \text{（区域 I）} \\
V_0, & 0 \le x \le a \quad \text{（区域 II）} \\
0, & x > a \quad \text{（区域 III）}
\end{cases}
$$

粒子从左侧入射，能量 $E < V_0$。经典力学中，粒子会被势垒完全挡回。量子力学中，粒子有一定的概率**隧穿**到势垒的另一侧。

**区域 I（$x < 0$）**：$V = 0$，波数 $k = \sqrt{2mE}/\hbar$。

$$
\psi_I(x) = A e^{i k x} + B e^{-i k x}
$$

**区域 II（$0 \le x \le a$）**：$V = V_0 > E$。波数 $q = \sqrt{2m(E - V_0)}/\hbar = i \kappa$，其中 $\kappa = \sqrt{2m(V_0 - E)}/\hbar > 0$。薛定谔方程的通解包含指数衰减和指数增长两项（因为区域 II 有限长，不需要在无穷远处舍去增长项）：

$$
\psi_{II}(x) = C e^{\kappa x} + D e^{-\kappa x}
$$

**区域 III（$x > a$）**：$V = 0$，波数仍为 $k$。只有向右传播的透射波：

$$
\psi_{III}(x) = F e^{i k x}
$$

#### 3.2 边界条件与待定系数的求解

在 $x = 0$ 和 $x = a$ 处，波函数及其导数必须连续（势能有有限跳跃）。

**在 $x = 0$ 处**：

波函数连续：$\psi_I(0) = \psi_{II}(0) \Rightarrow A + B = C + D$。

导数连续：$\psi_I'(0) = \psi_{II}'(0) \Rightarrow i k (A - B) = \kappa (C - D)$。

**在 $x = a$ 处**：

波函数连续：$\psi_{II}(a) = \psi_{III}(a) \Rightarrow C e^{\kappa a} + D e^{-\kappa a} = F e^{i k a}$。

导数连续：$\psi_{II}'(a) = \psi_{III}'(a) \Rightarrow \kappa (C e^{\kappa a} - D e^{-\kappa a}) = i k F e^{i k a}$。

我们关心的是透射系数 $T = |F|^2/|A|^2$（因为区域 I 和 III 的波数相同，概率流比等于振幅模方比）。

从后两个方程解出 $C$ 和 $D$ 用 $F$ 表达。将 $x = a$ 处的两个方程视为关于 $C e^{\kappa a}$ 和 $D e^{-\kappa a}$ 的线性方程组。写为：

$$
C e^{\kappa a} + D e^{-\kappa a} = F e^{i k a}
$$
$$
C e^{\kappa a} - D e^{-\kappa a} = \frac{i k}{\kappa} F e^{i k a}
$$

两式相加：$2 C e^{\kappa a} = F e^{i k a} \left(1 + \frac{i k}{\kappa}\right)$，因此：

$$
C = \frac{F}{2} e^{i k a} e^{-\kappa a} \left(1 + \frac{i k}{\kappa}\right)
$$

两式相减：$2 D e^{-\kappa a} = F e^{i k a} \left(1 - \frac{i k}{\kappa}\right)$，因此：

$$
D = \frac{F}{2} e^{i k a} e^{\kappa a} \left(1 - \frac{i k}{\kappa}\right)
$$

现在将 $C$ 和 $D$ 代入 $x = 0$ 处的边界条件。先算 $C + D$（$= A + B$）：

$$
\begin{aligned}
C + D &= \frac{F}{2} e^{i k a} \left[ e^{-\kappa a}\left(1 + \frac{i k}{\kappa}\right) + e^{\kappa a}\left(1 - \frac{i k}{\kappa}\right) \right] \\
&= \frac{F}{2} e^{i k a} \left[ (e^{\kappa a} + e^{-\kappa a}) - \frac{i k}{\kappa}(e^{\kappa a} - e^{-\kappa a}) \right] \\
&= F e^{i k a} \left[ \cosh(\kappa a) - i \frac{k}{\kappa} \sinh(\kappa a) \right]
\end{aligned}
$$

再算 $C - D$（与 $i k (A - B) = \kappa (C - D)$ 关联）：

$$
\begin{aligned}
C - D &= \frac{F}{2} e^{i k a} \left[ e^{-\kappa a}\left(1 + \frac{i k}{\kappa}\right) - e^{\kappa a}\left(1 - \frac{i k}{\kappa}\right) \right] \\
&= \frac{F}{2} e^{i k a} \left[ -(e^{\kappa a} - e^{-\kappa a}) + \frac{i k}{\kappa}(e^{\kappa a} + e^{-\kappa a}) \right] \\
&= F e^{i k a} \left[ -\sinh(\kappa a) + i \frac{k}{\kappa} \cosh(\kappa a) \right]
\end{aligned}
$$

现在利用 $x = 0$ 的两个边界条件消去 $A$ 和 $B$，求出 $F/A$。

由 $A + B = C + D$ 和 $i k (A - B) = \kappa (C - D)$ 可得：

$$
A = \frac{1}{2}\left[(C + D) + \frac{\kappa}{i k}(C - D)\right] = \frac{1}{2}\left[(C + D) - i \frac{\kappa}{k}(C - D)\right]
$$

代入 $C + D$ 和 $C - D$ 的表达式。经过代数运算（合并 $\cosh$ 和 $\sinh$ 项）：

$$
\begin{aligned}
A &= \frac{F}{2} e^{i k a} \left\{ \left[ \cosh(\kappa a) - i \frac{k}{\kappa} \sinh(\kappa a) \right] - i \frac{\kappa}{k} \left[ -\sinh(\kappa a) + i \frac{k}{\kappa} \cosh(\kappa a) \right] \right\} \\
&= \frac{F}{2} e^{i k a} \left\{ \cosh(\kappa a) - i \frac{k}{\kappa} \sinh(\kappa a) + i \frac{\kappa}{k} \sinh(\kappa a) + \cosh(\kappa a) \right\} \\
&= \frac{F}{2} e^{i k a} \left[ 2\cosh(\kappa a) + i \left( \frac{\kappa}{k} - \frac{k}{\kappa} \right) \sinh(\kappa a) \right]
\end{aligned}
$$

因此：

$$
\boxed{F = A \cdot \frac{2 e^{-i k a}}{2\cosh(\kappa a) + i \left( \frac{\kappa}{k} - \frac{k}{\kappa} \right) \sinh(\kappa a)}}
$$

透射系数 $T = |F|^2/|A|^2$。分母模方为：

$$
4\cosh^2(\kappa a) + \left( \frac{\kappa}{k} - \frac{k}{\kappa} \right)^2 \sinh^2(\kappa a)
$$

利用 $\cosh^2 x - \sinh^2 x = 1$，$\cosh^2 = 1 + \sinh^2$，以及 $\left( \frac{\kappa}{k} - \frac{k}{\kappa} \right)^2 = \frac{\kappa^2}{k^2} - 2 + \frac{k^2}{\kappa^2} = \frac{(\kappa^2 + k^2)^2}{k^2\kappa^2} - 4 = \frac{(k^2 + \kappa^2)^2}{k^2\kappa^2} - 4$。经过化简，最终得到：

$$
\boxed{T = \frac{1}{1 + \frac{1}{4}\left( \frac{\kappa}{k} + \frac{k}{\kappa} \right)^2 \sinh^2(\kappa a)}}
$$

#### 3.3 物理讨论

**厚势垒极限**（$\kappa a \gg 1$）：$\sinh(\kappa a) \approx \frac{1}{2} e^{\kappa a}$ 指数级大，分母中的 $1$ 可忽略。$\frac{1}{4}\left( \frac{\kappa}{k} + \frac{k}{\kappa} \right)^2 \frac{1}{4} e^{2\kappa a} \gg 1$。因此：

$$
\boxed{T \approx \frac{16 k^2 \kappa^2}{(k^2 + \kappa^2)^2} e^{-2\kappa a}}
$$

透射系数随势垒宽度 $a$ 指数衰减。这是量子隧穿的标志性特征——扫描隧道显微镜（STM）正是利用隧穿电流对针尖-样品距离的指数敏感性来实现原子级分辨率。

**薄势垒极限**（$\kappa a \ll 1$）：$\sinh(\kappa a) \approx \kappa a \ll 1$。分母中第二项为 $\frac{1}{4}(\frac{\kappa}{k} + \frac{k}{\kappa})^2 (\kappa a)^2 \ll 1$。$T \approx 1$ ——势垒几乎完全透明。

** $E = V_0$ 的单独处理**：此时 $\kappa = 0$，上述公式中的 $\sinh(\kappa a)/\kappa \to a$。取极限 $\kappa \to 0$，$\sinh(\kappa a) \to \kappa a$，$\frac{\kappa}{k} \to 0$，$\frac{k}{\kappa} \to \infty$。$T$ 公式退化为 $T = \frac{1}{1 + \frac{1}{4}(k/\kappa)^2 (\kappa a)^2} = \frac{1}{1 + k^2 a^2/4}$。这与直接从 $E=V_0$ 时 $\psi_{II}'' = 0$ 求解得到的结果完全一致。

#### 3.4 $E > V_0$：共振透射

当 $E > V_0$ 时，将 $\kappa$ 替换为 $i q$（其中 $q = \sqrt{2m(E - V_0)}/\hbar$），$\sinh(\kappa a)$ 变为 $i\sin(q a)$。透射系数公式变为：

$$
\boxed{T = \frac{1}{1 + \frac{1}{4}\left( \frac{q}{k} - \frac{k}{q} \right)^2 \sin^2(q a)}}
$$

当 $\sin(q a) = 0$ 时，$T = 1$ ——粒子完全透射，仿佛势垒不存在。这称为**共振透射**，发生在 $q a = n\pi$（$n = 1, 2, 3, \ldots$）时。物理图像是：入射波在势垒的两个界面之间来回反射，当势垒宽度恰好是半波长的整数倍时，多次反射波干涉相长，透射振幅达到极大值。这类似于光学中的法布里-珀罗干涉仪。


### 四、一维方势阱的散射共振

如果 $V_0 < 0$（势阱而非势垒），上述公式同样适用（只需将 $V_0$ 取负值）。此时 $q = \sqrt{2m(E + |V_0|)}/\hbar > k$。透射系数仍然呈现共振结构：当 $q a = n\pi$ 时 $T = 1$。

势阱散射共振的物理意义是：入射粒子在阱内形成暂态的准束缚态。这些准束缚态的能量对应 $T=1$ 的入射能量。在三维散射中，这类共振对应散射截面的 Breit-Wigner 峰。


### 五、S 矩阵的引入及其幺正性

#### 5.1 S 矩阵的定义

散射问题可以用**散射矩阵**（S 矩阵）统一表述。在渐近区，波函数可以写为向左和向右传播的平面波的叠加。将左侧和右侧的“入射”振幅与“出射”振幅通过 S 矩阵联系起来。

在势场的左侧，入射波（向右）振幅为 $A$，出射波（向左）振幅为 $B$。在势场的右侧，入射波（向左）振幅设为 $G$（对于从左侧入射的边界条件，$G = 0$），出射波（向右）振幅为 $F$。S 矩阵将“入射”振幅映射为“出射”振幅。定义：

$$
\begin{pmatrix} B \\ F \end{pmatrix} = \begin{pmatrix} S_{11} & S_{12} \\ S_{21} & S_{22} \end{pmatrix} \begin{pmatrix} A \\ G \end{pmatrix}
$$

其中 $S_{11}$ 是左侧入射到左侧出射的反射振幅，$S_{21}$ 是左侧入射到右侧出射的透射振幅。

对于一维对称势垒（$V(x) = V(-x)$），S 矩阵具有对称性：$S_{12} = S_{21}$（透射振幅对称），$S_{11} = S_{22}$（反射振幅对称）。

#### 5.2 幺正性

概率守恒要求 S 矩阵是幺正的：$S^\dagger S = I$。这等价于：

$$
|S_{11}|^2 + |S_{21}|^2 = 1, \quad |S_{12}|^2 + |S_{22}|^2 = 1, \quad S_{11}^* S_{12} + S_{21}^* S_{22} = 0
$$

第一个等式正是 $R + T = 1$（从左侧入射）。幺正性是散射理论中概率守恒的矩阵表述，在三维散射和量子场论中有着深远的推广。

一维散射问题概括起来就是：**连续谱的定态薛定谔方程在“出射波”边界条件下的解，给出了反射和透射振幅，其模方为可观测的散射概率**。从势阶到势垒再到势阱，从指数隧穿到共振透射，一维散射展示了量子力学中最基本的波动干涉效应，为理解更复杂的三维散射和量子输运现象奠定了坚实的基础。
