## 统计物理课程导航

这套课程从微观状态计数出发，依次建立微正则、正则和巨正则系综，再进入经典气体、量子气体、非平衡输运及现代专题。目录中的标题均链接到仓库内对应讲义；建议先完成基础主线，再按需要进入应用或前沿路线。

### 学习目标

完成基础主线后，读者应能：

1. 从宏观约束写出相应系综，并说明等概率或最大熵假设的适用范围；
2. 从配分函数推导 $U$、$S$、$p$、$\mu$、热容和涨落；
3. 判断经典、玻色和费米统计各自适用的温度与密度区间；
4. 用态密度、鞍点近似或输运方程复现至少一个完整物性计算；
5. 区分严格结果、热力学极限、弱耦合近似、分子混沌假设和前沿数值证据。

### 建议路线

- **基础主线**：[[数学准备]] → 第一章 → 第二章 → 第三章 → 第五章 5.1--5.6。
- **经典物性与输运**：基础主线 → 第四章 → 第八章。
- **量子与前沿**：基础主线 → 第六章 → 第七章 → 第九章。第九章包含研究层面主题，不作为前面结论的证明来源。

### 统一约定

| 符号        | 含义与约定                                  |
| --------- | -------------------------------------- |
| $k_B$     | 玻尔兹曼常数；温度采用 K，$\beta=(k_BT)^{-1}$      |
| $N,V,U$   | 粒子数、体积和内能；$n=N/V$ 为数密度                 |
| $p$       | 压强；动量在需要避免混淆时写作 $\mathbf p$            |
| $S$       | 热力学熵；量纲为 $\mathrm{J\,K^{-1}}$          |
| $Z,\Xi$   | 正则配分函数和巨配分函数，均为无量纲                     |
| $\mu$     | 化学势；$F=U-TS$，巨势记为 $\Phi_G=-k_BT\ln\Xi$ |
| $h,\hbar$ | 普朗克常数与约化普朗克常数，$h=2\pi\hbar$            |

除非文章另行声明，经典相空间积分采用 SI 制和 $1/(N!h^{3N})$ 归一化；量子迹对物理态求和，简并度不得重复计数。连续谱、热力学极限、负温度和强耦合平衡态均需查看对应文章中的额外条件。

## 预备章：[[数学准备]] [数学准备 - 知乎](https://zhuanlan.zhihu.com/p/2061203392161227384)

在正式进入统计物理之前，集中处理贯穿全课程的核心数学工具，避免在各章推导中反复打断主线。

- **斯特林公式**：$N! \sim \sqrt{2\pi N} (N/e)^N$，包括其渐近展开和误差估计。统计物理中几乎所有大数近似都依赖此公式。
- ** $N$ 维球体积**：$\Gamma$ 函数的引入，$V_N(R)$ 的完整推导，及其在相空间积分中的应用。
- **高斯积分**：从一维到多维，含二次型的对角化技巧。
- **鞍点近似与最速下降法**：形如 $\int e^{N f(x)} dx$ 的积分在大 $N$ 极限下的近似，是经典极限和平均场理论的核心数学方法。
- **拉格朗日乘子法**：约束优化问题的标准解法，在最大熵原理和系综推导中反复使用。

## 第三章：正则系综与配分函数——热库下的统治性工具

本章引入统计物理中最强大、最通用的计算框架——正则系综，构建从微观配分函数直达宏观自由能的“高速公路”。

**[[3.1 系统与热库的耦合——玻尔兹曼因子的严格推导]]** [3.1 系统与热库的耦合——玻尔兹曼因子的严格推导 - 知乎](https://zhuanlan.zhihu.com/p/2061208548953133080)
- 物理设定：系统 $S$ 与大热库 $R$ 接触，$T, V, N$ 固定
- 将系统+热库视为孤立系统，总能量 $E_{\text{tot}} = E_S + E_R$ 守恒
- 将热库熵 $S_R(E_{\text{tot}} - E_S)$ 在 $E_S$ 附近泰勒展开，保留线性项
- 导出系统处于微观态 $i$（能量 $E_i$）的概率：$P_i \propto \Omega_R(E_{\text{tot}} - E_i) \propto e^{S_R(E_{\text{tot}} - E_i)/k_B} \propto e^{-E_i / (k_B T)}$

**[[3.2 正则配分函数Z的定义与归一化|3.2 正则配分函数 $Z$ 的定义与归一化]]** [3.2 正则配分函数Z的定义与归一化 - 知乎](https://zhuanlan.zhihu.com/p/2061208654309744912)
- 离散能级：$Z = \sum_i e^{-\beta E_i}$，$\beta \equiv 1/(k_B T)$
- 经典连续形式：$Z = \frac{1}{N! h^{3N}} \int d^{3N}p \, d^{3N}q \, e^{-\beta H(p,q)}$
- $Z$ 作为归一化分母：$P_i = e^{-\beta E_i} / Z$
- 当能级存在简并时的推广：$Z = \sum_i g_i e^{-\beta E_i}$

**[[3.3 亥姆霍兹自由能的枢纽作用|3.3 亥姆霍兹自由能 $F=-k_BT\ln Z$ 的枢纽作用]]** [3.3 亥姆霍兹自由能的枢纽作用 - 知乎](https://zhuanlan.zhihu.com/p/2061208817954723473)
- 内能 $U$ 的统计表达：$U = \sum_i P_i E_i$
- 熵的统计表达：$S = -k_B \sum_i P_i \ln P_i$（吉布斯熵公式）
- 证明 $F \equiv U - TS = -k_B T \ln Z$：配分函数与热力学势的精确等价
- 通过全微分验证 $dF = -S dT - p dV + \mu dN$：确立 $F$ 作为“母函数”的地位

**[[3.4 所有热力学量的“批发式”求导]]** [3.4 所有热力学量的“批发式”求导 - 知乎](https://zhuanlan.zhihu.com/p/2061208893762679901)
- 内能：$U = -\frac{\partial \ln Z}{\partial \beta}$
- 压强：$p = \frac{1}{\beta} \frac{\partial \ln Z}{\partial V}$
- 熵：$S = k_B \left( \ln Z + \beta U \right) = -\left( \frac{\partial F}{\partial T} \right)_V$
- 化学势：$\mu = -k_B T \left( \frac{\partial \ln Z}{\partial N} \right)_{T,V}$
- 热容：$C_V = \left( \frac{\partial U}{\partial T} \right)_V$

**[[3.5 能量涨落与热力学稳定性的微观起源]]** [3.5 能量涨落与热力学稳定性的微观起源 - 知乎](https://zhuanlan.zhihu.com/p/2061208963824275893)
- 能量二次矩的计算：$\overline{E^2} = \frac{1}{Z} \frac{\partial^2 Z}{\partial \beta^2}$
- 推导涨落公式：$\Delta E^2 = \overline{E^2} - \overline{E}^2 = \frac{\partial^2 \ln Z}{\partial \beta^2} = k_B T^2 C_V$
- 对具有短程关联且远离临界点的广延系统，相对涨落通常满足 $\Delta E/U\sim N^{-1/2}\to0$
- 说明系综等价的常见条件，并指出临界点、长程相互作用或 $U$ 接近零时需要单独分析

**[[3.6 量子-经典对应中 $h$ 因子的最终确立]]** [3.6 量子-经典对应中 $h$ 因子的最终确立 - 知乎](https://zhuanlan.zhihu.com/p/2061209025253994784)
- 半经典态计数：相空间测度除以 $h^{3N}$；它给出经典极限的态数归一化，不要求选定物理格点
- 半经典配分函数：$Z = \frac{1}{N! h^{3N}} \int d\Gamma \, e^{-\beta H}$
- 何时必须使用量子配分函数：热波长 $\lambda_{\text{dB}} \equiv h/\sqrt{2\pi m k_B T}$ 与粒子间距可比时


## 第四章：经典统计力学的典范应用——从微观运动到宏观物性

本章将正则系综应用于各种经典物理模型，系统解释日常热学现象与物态方程。

**[[4.1 能量均分定理的完整证明]]** [4.1 能量均分定理的完整证明 - 知乎](https://zhuanlan.zhihu.com/p/2061209559172224780)
- 将哈密顿量分解为广义动量的二次型：$H = \sum_i \frac{p_i^2}{2m_i} + V(q)$
- 利用高斯积分证明每个二次方自由度对配分函数的贡献为 $\sqrt{2\pi m/\beta}$
- 导出每个二次方自由度平均分得 $\frac{1}{2} k_B T$ 的热能
- 适用范围讨论：仅对二次方项成立；要求经典极限；非二次型势能不服从均分

**[[4.2 爱因斯坦固体与杜隆-珀蒂定律]]** [4.2 爱因斯坦固体与杜隆-珀蒂定律 - 知乎](https://zhuanlan.zhihu.com/p/2061209648745785052)
- 将含 $N$ 个原子的固体近似为 $3N$ 个独立的一维谐振模式（等价于 $N$ 个三维振子）
- 计算经典配分函数得到 $U = 3N k_B T$

- 在分子混沌假设下严格证明：$dH/dt \le 0$
- 熵与 $H$ 的关系：在固定粒子数并选定相空间测度归一化后，动力学熵与 $-k_BH$ 相差常数，因此 $dS/dt=-k_BdH/dt\ge0$
- **洛施密特佯谬的形式化解**：$H$ 定理的推导依赖于分子混沌假设，这个假设本身引入了时间的方向性——碰撞前的粒子不关联，碰撞后产生关联。不可逆性不是来自微观动力学，而是来自我们对初始条件的特殊选择
- 熵增如何从时间反演对称的动力学中“涌现”：宏观系统的演化从低概率态走向高概率态，而“低概率”状态在时间反演下仍然低概率——时间之矢是统计优势的方向

**[[8.4 弛豫时间近似与输运系数的导出]]** [8.4 弛豫时间近似与输运系数的导出 - 知乎](https://zhuanlan.zhihu.com/p/2061211698418169609)
- 将碰撞项近似为线性弛豫：$\left( \frac{\partial f}{\partial t} \right)_{\text{coll}} \approx -\frac{f - f_0}{\tau}$
- 在外场（电场、温度梯度）下求解玻尔兹曼方程得到非平衡分布修正
- 电导率的微观推导：$\sigma = \frac{n e^2 \tau}{m}$（德鲁德模型结果）
- 热导率与黏度的微观表达式
- 维德曼-弗兰兹定律：$\kappa/(\sigma T) = \frac{\pi^2}{3} (k_B/e)^2$（索末菲值）

**[[8.5 涨落-耗散定理与爱因斯坦关系]]** [8.5 涨落-耗散定理与爱因斯坦关系 - 知乎](https://zhuanlan.zhihu.com/p/2061211757960488671)
- 建立平衡态热涨落（自发噪声）与线性响应（耗散）之间的严格关联
- 爱因斯坦关系：扩散系数与机械迁移率 $\mu_{\mathrm{mob}}$ 满足 $D=\mu_{\mathrm{mob}}k_BT$，避免与化学势 $\mu$ 混淆
- 奈奎斯特公式：电阻中的热噪声电压谱密度
- 涨落-耗散定理的普遍形式：线性响应函数（耗散部分）与平衡态关联函数（涨落）之间的解析关系

**[[8.6 布朗运动与朗之万方程]]** [8.6 布朗运动与朗之万方程 - 知乎](https://zhuanlan.zhihu.com/p/2061211827904816430)
- 牛顿方程中引入随机力与阻尼项：$m \dot{v} = -\gamma v + \eta(t)$
- 随机力的统计性质：$\langle \eta(t) \rangle = 0$，$\langle \eta(t) \eta(t') \rangle = \Gamma \delta(t-t')$
- 涨落-耗散关系：$\Gamma = 2\gamma k_B T$
- 长时间极限 $t\gg m/\gamma$ 下，一维均方位移为 $\langle[x(t)-x(0)]^2\rangle=2(k_BT/\gamma)t$（正常扩散）
- 从朗之万方程到福克-普朗克方程：概率分布 $P(x, v, t)$ 的演化方程


## 第九章：现代前沿专题——信息、临界与量子热化

本章作为全课程的升华，探讨统计物理在信息论、复杂系统及量子基础问题中的前沿应用。

**[[9.1 麦克斯韦妖与兰道尔擦除原理]]** [9.1 麦克斯韦妖与兰道尔擦除原理 - 知乎](https://zhuanlan.zhihu.com/p/2061211943252267442)
- 麦克斯韦妖的悖论：智能小妖通过获取分子信息区分快慢分子，在不做功的情况下降低系统熵——似乎违反了热力学第二定律
- 信息熵与热力学熵的统一：小妖获取的信息必须存储在物理记忆中
- **兰道尔擦除原理**（1961）：擦除 1 bit 信息必然耗散最少 $k_B T \ln 2$ 的能量，产生至少 $k_B \ln 2$ 的熵增——信息擦除是不可逆操作
- 信息的物理实在性：信息不是抽象的数学概念，而是与能量和熵一样真实的物理量
- 现代实验验证：2012 年 Bérut 等人用光镊实现单比特擦除，直接测量到兰道尔极限

**[[9.2 非平衡涨落定理]]** [9.2 非平衡涨落定理 - 知乎](https://zhuanlan.zhihu.com/p/2061212000332657466)
- Jarzynski 等式（1997）：$\langle e^{-\beta W} \rangle = e^{-\beta \Delta F}$——无论过程多么远离平衡，功的指数平均精确等于自由能差的指数
- Crooks 涨落定理（1999）：$P_F(W) / P_R(-W) = e^{\beta(W - \Delta F)}$——正向与反向过程中功分布之比
- 单分子操控实验的验证：RNA 发夹的拉伸-回缩循环
- 这些定理的深刻意义：热力学第二定律（$\langle W \rangle \ge \Delta F$）是 Jarzynski 等式的 Jensen 不等式推论——第二定律是统计规律的必然结果，不是独立的原理

**[[9.3 临界现象与重整化群（RG）的定性思想]]** [9.3 临界现象与重整化群（RG）的定性思想 - 知乎](https://zhuanlan.zhihu.com/p/2061212060533404814)
- 铁磁相变中的序参量（自发磁化 $M$）与关联函数：$\langle \sigma_i \sigma_j \rangle \sim e^{-|i-j|/\xi}$
- 在临界点，关联长度 $\xi$ 发散——所有尺度都同等重要，涨落存在于从微观到宏观的所有尺度
- **标度律**：热力学量在临界点附近表现为约化温度 $t = (T - T_c)/T_c$ 的幂律，如 $C \sim |t|^{-\alpha}$，$M \sim |t|^\beta$，$\xi \sim |t|^{-\nu}$
- 普适类：不同微观系统可共享临界指数；分类通常取决于空间维数、序参量对称性、相互作用程和守恒动力学等相关特征
- 重整化群的粗粒化策略：对系统进行尺度变换，将小尺度涨落积分掉，观察有效耦合常数的“流”
- 临界点是 RG 变换的不稳定不动点

**[[9.4 伊辛模型的精确解与平均场近似]]** [9.4 伊辛模型的精确解与平均场近似 - 知乎](https://zhuanlan.zhihu.com/p/2061212144457326957)
- 伊辛模型：$\hat{H} = -J \sum_{\langle i,j \rangle} \sigma_i \sigma_j - h \sum_i \sigma_i$（$\sigma_i = \pm 1$）
- **一维伊辛模型的转移矩阵严格解**：$Z = \text{Tr}(T^N)$，$T$ 为 $2 \times 2$ 转移矩阵；$T_c = 0$，有限温度无相变
- **二维伊辛模型的昂萨格解**（1944）的意义：存在有限温度相变 $T_c = 2 J/(k_B \ln(1+\sqrt{2}))$，比热对数发散 $\alpha = 0$——相变统计力学的第一个严格解
- 平均场近似（Weiss 分子场理论）：将每个自旋感受到的环境等效为平均场 $h_{\text{eff}} = J z \langle \sigma \rangle + h$；自洽方程 $\langle \sigma \rangle = \tanh(\beta J z \langle \sigma \rangle)$；居里温度 $T_c^{\text{MF}} = J z / k_B$
- 平均场理论的临界指数：$\beta = 1/2, \gamma = 1, \delta = 3$——与真实二维伊辛模型不同，但在四维以上空间精确成立（上临界维数 $d_c = 4$）

**[[9.5 孤立量子系统的热化——本征态热化假说（ETH）]]** [9.5 孤立量子系统的热化——本征态热化假说（ETH） - 知乎](https://zhuanlan.zhihu.com/p/2061212203160712069)
- 经典统计力学的基础是各态历经假说；封闭量子系统的“各态历经”是什么？
- 封闭量子系统在幺正演化下不会达到混态——那么热化意味着什么？
- 热化 = 对于少数体可观测量的期望值，长时间平均等于微正则系综平均
- **本征态热化假说**（ETH）：能量本征态的可观测量期望值在窄能量窗口内光滑变化；这意味着单个本征态已经蕴含了热力学信息
- ETH 的解释：初始纯态的非平衡演化在可观测量意义上无法与平衡态区分——去相位是表观热化的机制
- ETH 成立的证据与条件

**[[9.6 多体局域化（MBL）及其对统计力学的挑战]]** [9.6 多体局域化（MBL）及其对统计力学的挑战 - 知乎](https://zhuanlan.zhihu.com/p/2061212259439977536)
- 无序如何抑制热化：安德森局域化在无相互作用系统中的推广——有相互作用的多体系统是否也会局域化？
- **多体局域化候选相**：理想化强无序模型可出现非遍历本征态并长期保留初态记忆；高维和热力学极限中的稳定性仍受热泡与雪崩机制挑战
- MBL 的候选标志：量子猝冷后纠缠熵随时间对数增长；高能本征态呈面积律而热化本征态呈体积律；并出现准局域守恒量（LIOMs）
- MBL 作为 ETH 失效机制：常规热系综不能描述其全部局域守恒量，但这不等于所有统计描述都失效
- 证据边界：区分有限尺寸数值、有限时间实验和严格热力学极限结论
